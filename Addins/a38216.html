Подробное описание

Класс входит в Проект Properties
Класс команды проверки существования файлов страниц свойств, общих для всех проектов решения.

  • При выполнении команда выводит в лог список существующих файлов страниц свойств в папке общих страниц свойств для всех проектов решения.
  • Путь к папке страниц свойств извлекается из настроек (параметр PathToSolutionPropsFile раздела VisualStudio).
