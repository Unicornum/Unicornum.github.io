Подробное описание

Класс входит в Раздел Utilities
Класс доступа к статистике работы над всеми решениями, которые были изменены за последние две недели.

  • Номера недель отсчитываются от 1 января 1900 года (1...7 января 1900 - нулевая неделя).

Конструктор(ы)

Info_t::AllSolutionsDevelopmentInfo ( const Path_t &  _PathToSourceFile,
const Time_t &  _SavePeriod,
const Time_t &  _IgnoreTime 
)

Конструктор класса.

  • Загружает информацию за две последние недели из указанного файла.
  • Если файла не существует, текущая информация будет содержать 'нулевые' значения.
  • Если указанный файл не существует, будет создана пустая структура.
Аргументы
[in]_PathToSourceFileПуть к файлу, из которого будет загружена информация.
[in]_SavePeriodПериод времени, через который будет производиться сохранение текущего состояния в файл (при вызове AddTime()).
[in]_IgnoreTimeПромежуток времени между соседними действиями, который будет игнорироваться при подсчете времени работы над решением.
Исключения
std::exception
  • Невозможно открыть указанный файл.
  • Указанный файл имеет некорректную структуру.

void Info_t::UpdateTime ( const SolutionName_t &  _SolutionName)

Функция обновления времени существующих данных.

  • Время обновляется текущему дню текущей недели.
  • Если для указанного решения нет записей, создается набор элементов со значениями по умолчанию.
  • Функция производит сохранение изменений через промежутки времени, заданные в конструкторе.
Аргументы
[in]_SolutionNameИмя решения, которому будет добавлено время.
