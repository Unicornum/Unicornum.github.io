Класс входит в ConnectionCommands
Класс выполнения команды, не связанной с пунктом меню.

Параметры шаблона
TUsefulCommandКласс команды, которая должна быть выполнена.
  • В конструкторе должна получать указатель на const utilities::vs::Data.
  • Логика ее действий должна содержаться в функции void Execute(void).
TDataКласс данных, которые должны быть переданы классу исполняемой команды.
