Методы

void __stdcall SolutionEvents::ProjectAdded ( EnvDTE::Project *  _pProject)

Функция вызывается после загрузки проекта.

Вызывается:

  • При создании нового проекта.
  • При подключении существующего проекта (вручную или надстройкой).
  • При загрузке проектов для папки решения.
  • VS2013: для каждого проекта при загрузке решения.
Аргументы
[in]_pProjectУказатель на объект проекта, который загружается.

Граф вызовов:

void __stdcall SolutionEvents::ProjectRemoved ( EnvDTE::Project *  )

Функция вызывается перед выгрузкой проекта.

Вызывается:

  • При удалении проекта.
  • При выгрузке проектов из папки решения.
