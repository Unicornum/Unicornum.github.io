Подробное описание

Класс входит в Проект Documentation
Класс команды добавления описания функции/класса.

  • При выполнении команды курсор должен стоять на строке функции/класса, для которой следует добавить описание.
  • Если строка, на которой установлен курсор не содержит определения класса или реализации функции, команда будет неактивна.
  • Добавление шаблона описания производится в ближайшую пустую строку перед текущей.

Функция получения текста, отображаемого в строке команды.

  • Каждый класс-наследник должен переопределить функцию и определить текст, отображаемый для текущей команды.
  • По умолчанию возвращает имя класса.

Переопределяет метод предка core::useful_commands::common::IMenuCommand.
