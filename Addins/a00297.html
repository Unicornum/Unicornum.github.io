Подробное описание

Класс входит в Проект Dialog
Класс диалогового окна для отображения процесса на базе MFC.

  • Отображает модальное диалоговое окно с прогресс-баром.
  • Регулярно вызывает функцию DoProcess() объекта, переданного в конструкторе, устанавливая прогресс-бару значение, возвращаемое этой функцией.
  • Завершается с кодом IDOK, когда объект, переданный в конструкторе возвращает true при вызове функции IsEnded().
  • Завершается после нажатия кнопки закрытия окна с кодом IDCANCEL.
  • Может работать как с отображением прогресса в %, так и в качестве 'бегущей строки' (режим задается флагом конструктора).
Заметки
Для того, чтобы можно было создавать диалоговое окно, объявленное в ресурсах .lib, в свойствах .dll в разделе 'Компоновщик\Общие' поставить параметру 'Использовать входные данные библиотек, указанных в зависимостях' значение 'Да'.
