Подробное описание

Класс входит в Проект Documentation
Класс команды генерации простого подставного класса.

\

  • Подставной класс генерируется для класса, на котором стоит курсор в файле в той же папке, что и активный документ с именем, совпадающем с именем исходного документа, но со вставкой '.mock' (например для файла Example.hpp будет сгенерирован файла Example.mock.hpp).
  • Команда доступна, если курсор стоит на объявлении класса и отсутствует файл с тем именем/расширением со вставкой '.mock'.

Функция получения текста, отображаемого в строке команды.

  • Каждый класс-наследник должен переопределить функцию и определить текст, отображаемый для текущей команды.
  • По умолчанию возвращает имя класса.

Переопределяет метод предка core::useful_commands::common::IMenuCommand.
