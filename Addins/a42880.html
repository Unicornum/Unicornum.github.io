Подробное описание

Класс входит в RegexTester
Интерфейс для записи данных модели в объект представления.

  • Используется при запуске программы.
  • Поскольку при запуске программы активны будут вкладка Match и 'Исходные строки', функции передачи других параметров не нужны.
