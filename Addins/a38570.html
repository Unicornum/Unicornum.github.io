bool DocumentImpl::ForClassUnderCursorExistTestProject ( void  ) const
virtual

Функция проверки существования тестового проекта для класса, на объявлении которого стоит курсор.

  • Предполагается, что при вызове функции курсор гарантированно стоит на объявлении класса, поэтому перед вызовом этой функции следует вызывать функцию ContainsClassDeclarationUnderCursor().
Возвращает
true У класса есть тестовый проект.
false У класса нет тестового проекта.

Замещает wrappers::helper::IDocument.
