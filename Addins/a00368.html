CString Text::GetClassName ( const CString &  _Text)
static

Функция для извлечения имени класса из текста.

  • Для текста 'class ClassName : public Base' функция вернет 'ClassName'.
  • Предварительное объявление класса ('class ClassName;') не считается валидным исходным текстом.
Аргументы
[in]_TextИсходный текст.
Исключения
std::exception
  • Строка не содержит объявления класса.
  • Строка содержит предварительное объявление класса.

Функция для извлечения имени функции из текста.
