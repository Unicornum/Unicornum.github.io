◆ Check()

CString Doxygen::Check ( const CString &  _Text)
overridevirtual

Функция для фильтрации сообщений, получаемых от Doxygen.

  • Предназначена для фильтрации строк, получаемых от Doxygen.
  • Для строк, несущих информативный характер, возвращает пустую строку.
  • Игнорирует warning'и, которые относятся к вспомогательным тегам, используемым надстройкой.
  • Все error'ы возвращаются 'как есть'.
Аргументы
[in]_TextИсходная строка.
Возвращает
Результат обработки.

Замещает core::utilities::filter::IFilter.

◆ GetProgress()

int Doxygen::GetProgress ( void  ) const
overridevirtual

Функция для получения текущего прогресса обработки.

  • Значение прогресса обработки должно вычисляться в функции Check() при обработке получаемых строк.
  • Функция будет использоваться только в при использовании диалогового окна, отображающего прогресс обработки в процентах (для окна, отображающего 'бегущую строку' эта функция может просто возвращать 0).
Возвращает
Значение текущего прогресса обработки в процентах.
