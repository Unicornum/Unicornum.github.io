Подробное описание

Класс входит в Проект Project
Класс-фасад для списка проектов.

  • Представляет собой список папок и проектов родительского элемента (Solution или Project).

Пример использования:

// Проект, выделенный в дереве решения.
m_DteHost.Solution.Projects[Active];
// Добавление проекта в корень решения.
m_DteHost.Solution.Projects.Add(PathToTestAll);
// Добавление проекта в папку решения существующего проекта.
ExistProject.ParentFolder.Projects.Add(PathToTestProject);

Project Projects::operator[] ( const CString &  _Name) const

Поиск проекта по имени.

  • Поиск проекта производится для текущего объекта (без учета дочерних), т.е. для решения - только среди проектов в корне решения, для папки - только среди проектов этой папки.
Исключения
std::exception
  • Проекта с указанным именем в решении/папке не существует.

Project Projects::operator[] ( const Project::UniqueName &  _Name) const

Поиск проекта по уникальному имени.

  • Поиск проекта производится только для решения и по всему решению (рекурсивно).
Исключения
std::exception
  • Проекта с указанным именем в решении не существует.
