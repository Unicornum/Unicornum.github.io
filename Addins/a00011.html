Описание создания VSPackage на С++

Создание проекта

При создании проекта расширения создаются 3 проекта:

  • VSPackage1 - основной модуль расширения. Ключевой класс - CVSPackage1Package.
  • VSPackage1UI - вспомогательная dll (задается в файле .pkgdef как
    [$RootKey$\Packages\{f521d44a-05d4-4cf6-bbec-2a47fb6eadab}\SatelliteDll]
    "DllName"="VSPackage1UI.dll"
    "Path"="$PackageFolder$"
    ). Содержит иконки (в формате png) и структуру добавляемых команд (которую можно игнорировать?).
  • VSPackage1VSIX (C#). Этот проект, судя по всему, нужен только для хранения файла source.extension.vsixmanifest и создания файла .vsix (см. описание формата ниже).

Описание формата .vsix

Файл формата .vsix - это zip архив ('ручная' упаковка файлов работает, да), содержащий файлы:

  • VSPackage1.dll (главный модуль расширения).
  • VSPackage1.pkgdef (регистрация главного модуля расширения и вспомогательной dll).
  • extension.vsixmanifest (информация о расширении, которую будет отображать установщик VSIX и сама Visual Studio).
  • [Content_Types].xml (список расширений файлов, которые будет копировать установщик VSIX в папку результата при установке расширения).
  • 1033\VSPackage1UI.dll (вспомогательная dll главного модуля расширения).

Автозапуск расширения при старте Visual Studio

Добавить в файл .pkgdef строки

[$RootKey$\AutoLoadPackages\{ADFC4E64-0397-11D1-9F4E-00A0C911004F}]
"{CURRENT_PACKAGE_GUID}"=dword:00000000
@="UICONTEXT_NoSolution"

Получение объекта DTE

Использовать в классе C<PackageName>Package следующий код (в конструкторе при автостарте не сработает!):

CComPtr<IServiceProvider> pSP;
HRESULT hr1 = this->GetSite(&pSP);
if (FAILED(hr1)) MessageBox(NULL, _T("Failed1"), NULL, MB_OK);
CComPtr<EnvDTE::_DTE> m_pDTE;
HRESULT hr2 = pSP->QueryService(SID_SDTE, IID__DTE, (void **)&m_pDTE);
if (FAILED(hr2)) MessageBox(NULL, _T("Failed2"), NULL, MB_OK);

Для получения объекта DTE при старте Visual Studio добавить в класс CVSPackage1Package функцию:

// Вызывается при старте Visual Studio
void PostSited(IVsPackageEnums::SetSiteResult)
{
// ...
}

Завершение работы расширения:

// Вызывается перед закрытием Visual Studio
void PreClosing(void)
{
// ...
}
