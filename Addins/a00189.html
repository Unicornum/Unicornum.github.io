Подробное описание

Класс входит в Documentation
Класс команды увеличения номера версии класса.

  • Команда будет активна, если в момент ее вызова курсор стоит на строке объявления класса.
  • В раздел version добавляется версия класса, с увеличенным номером x.0.0.0.
  • В раздел data добавляется строка с текущей датой.
  • Добавляется/увеличивается год в разделе copyright.
