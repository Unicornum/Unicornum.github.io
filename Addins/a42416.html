virtual void model::IWriteTask::CreateChild ( size_t _Id)
pure virtual

Функция добавления дочерней задачи для указанной.

  • Новая задача должна быть добавлена в конец списка дочерних задач указанной задачи.
Аргументы
[in]_IdИдентификатор задачи, в список дочерних которой будет добавлена задача.
Исключения
std::exception
  • Задачи с указанным идентификатором не существует.

Функция переноса текущей задачи в другую родительскую задачу.

  • При переносе задача должна быть добавлена в конец спсика указнной родительской задачи.
Аргументы
[in]_IdИдентификатор переносимой задачи.
[in]_ParentIdИдентификатор родительской задачи, в которую должна быть перенесена текущая.
Исключения
std::exception
  • Задач с указанными идентификаторами не существует.

Замещается в model::AllTasks.

◆ SetPosition()

virtual void model::IWriteTask::SetPosition ( size_t _Id,
size_t _Position )
pure virtual

Функция изменения позиции текущей задачи.

  • Если позиция превышает количесво дочерних задач у родительской задачи текущей, переместить задачу в конец списка.
Аргументы
[in]_IdИдентификатор переносимой задачи.
[in]_PositionПозиция в списке родительской задачи, в которую должна быть установлена текущая (отсчет - от нуля).
Исключения
std::exception
  • Задачи с указанным идентификатором не существует.

Замещается в model::AllTasks.
