Подробное описание

Класс входит в ProjectTasks
Класс для работы с историей изменений решения.

  • История изменений формируется в виде, пригодном для генерации документации при помощи Doxygen (идентификатор для включения страницы в документацию - HistoryPage).
  • Позволяет добавлять сообщения о выполненных задачах с указанием даты/времени записи.
  • Позволяет добавлять разделы/подразделы (например, при изменении версии решения).
  • При превышении рамером файла истории изменений производит ротацию файлов, добавляя в конец файла ссылку на предыдущий.

Конструктор(ы)

◆ SolutionHistoryPage()

SolutionHistoryPage::SolutionHistoryPage ( const Data_t & _Data)
explicit

Конструктор класса.

  • Загружает файл истории изменений из параметра m_PathToSolutionHistoryFile (если файла нет, создает заготовку истории изменений).
  • Если размер файла m_PathToSolutionHistoryFile превышает m_MaxHistoryFileSize, производится ротация файлов истории изменений решения (текущий файл переименовывается с добавлением очередного индекса и создается новый файл со ссылкой на предыдущий).
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).

void SolutionHistoryPage::AddMessage ( const String_t & _Text)

Функция добавления записи в историю изменений решения.

  • Если запись производится для того же проекта, что и предыдущая, будет просто добавлена новая запись, в противном случае дополнительно будет добавлена запись с именем нового проекта.

Исключения
