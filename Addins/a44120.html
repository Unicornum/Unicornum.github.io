Конструктор(ы)

◆ ProjectInfo()

ProjectInfo::ProjectInfo ( const Path_t & _PathToProjectFile)
explicit

Конструктор класса.

  • Функция ожидает получить путь в формате {SolutionPath}\Solution\Solution\HighPart\...\LowPart\ProjectName\ProjectName\ProjectName.{Extension}, из которого будет извлекать информацию.
Аргументы
[in]_PathToProjectFileПуть к файлу проекта (вместе с самим файлом).
Исключения
std::exception
  • Полученный путь имеет неожиданный формат.

◆ GetFullName()

ProjectInfo::String_t ProjectInfo::GetFullName ( void ) const

Функция получения имени проекта внутри решения (включая имена разделов).

  • Имена разделов будут разделены '::'.
  • Для примера из конструктора результат будет выглядеть как "HighPart::...::LowPart::ProjectName".
