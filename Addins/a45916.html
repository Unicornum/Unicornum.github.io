Manual

Руководство пользователя

Продукт представляет собой расширение (надстройку) для Visual Studio, состоящее из отдельных модулей и предназначеное для:

  • Упрощения работы над кодом.
  • Автоматизации вспомогательных действий.
  • Выполнения рутинных операций.
  • Упрощения использования сторонних утилит.
Заметки
Текущую версию продукта (для отправки отчета об ошибках) можно посмотреть в главном меню (UniCA -> Версия: ...).

Вывод

Информационные сообщения и сообщения об ошибках при работе модулей выводятся в раздел 'UniCA' окна 'Вывод' Visual Studio.

Настройка продукта

Настройка продукта производится при помощи утилиты редактирования настроек, которая вызывается через главное меню Visual Studio (UniCA -> Параметры...).

Настройки продукта организованы в иерархическое дерево, корневой раздел которого содержит разделы версий Visual Studio.

Разделы версий Visual Studio наследуют параметры раздела Unknown (наследование в данном случае означает, что при изменении значения параметра из базового раздела изменятся и все значения этого параметра в унаследованных разделах, при этом изменение значения параметра конкретного раздела не затронет остальные).

Описание параметров доступно при выборе параметра мышью, описание параметров, используемых командами можно посмотреть в описании команд, их использующих.

Заметки
Изменение параметров вступит в силу при следующем запуске утилиты изменения настроек.

Макросы

Значения параметров могут содержать макросы (имя, заключенное в фигурные скобки), в качестве которых можно использовать имена параметров этого же или родительских разделов, кроме того, существуют предопределенные макросы:

  • SolutionPath - полный путь к папке текущего решения.
  • ModuleLocation - полный путь к папке расположения исполняемого файла текущего модуля.
  • ModuleDataLocation - полный путь к папке расположения данных текущего модуля (папка располагается внутри папки текущего пользователя Windows и уникальна для каждого пользователя).
