Класс входит в RegexTester
Базовый класс панели, содержащей выпадающее меню для поля ввода регулярного выражения.

  • Содержит объект выпадающего меню и функции-обработчики событий выбора строк меню.
  • Содержит TextCtrl нулевого размера, к которому привязано меню для того, чтобы оно не появлялось по всему окну при клике правой кнопкой мыши.
