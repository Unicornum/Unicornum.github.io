Класс входит в Testing
Класс для создания/подключения тестового файла класса.

  • В конструктор передается объект, содержащий информацию о классе, для которого будет создан файл теста.

◆ Create()

void TestClassBlank::Create ( const DTE_t & _DTE,
const SolutionFolder_t & _MainProjectFolder ) const

Создание тестового файла класса.

  • Тестовый файл будет подключен к проекту тестирования проекта (в папку/фильтр 'Tests\имя_класса').
  • Тестовый файл будет подключен к проекту тестирования класса.
  • Тестовый файл будет открыт в VisualStudio (станет активным документом).
Аргументы
[in]_DTEОбъект среды разработки.
[in]_MainProjectFolderПапка решения, в которой располагается проект, которому принадлежит класс, для которого создается тестовый файл.
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).
