Подробное описание

Класс входит в Проект Helper Вспомогательный класс для поиска страницы свойств у проекта.

  • Объект проекта, у которого будет производиться поиск передается в конструкторе.
  • Запуск поиска производится вызовом одной из открытых функций.
  • Поиск производится по всем конфигурациям проекта.
  • Для каждой найденной страницы свойств будет вызвана функция WasFoundNext() (класс-наследник должен определить ее).
  • Если у конфигурации нет указанной страницы свойств, будет вызвана функция WasNoFound() (класс-наследник должен определить ее).
