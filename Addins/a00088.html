Класс входит в ProjectTasks
Класс, содержащий полный список всех задач проекта.

  • В конструкторе создает корневые объекты задач для того, чтобы контроллер мог связать корневые элементы с панелью задач представления до загрузки списка задач из файла.

Конструктор(ы)

AllTasks::AllTasks ( const Data_t &  _Data)
explicit

Конструктор класса.

  • Корневые задачи разделов хранятся как дочерние элементы корневых элементов, которым не соответствуют никакие объекты задач.
  • Создает корневую задачу для задач, находящихся в процессе работы и создает ей дочерние задачи, загружая их из указанного xml файла (из узла 'Tasks\Process').
  • Создает корневую задачу для выполненных задач, создает ей дочерние задачи (для заданного количества недель, начиная с текущей) и создает им дочерние задачи, загружая их из указанного xml файла (из узла 'Tasks\Complete\Week####').
  • Если указанный файл не существует, будут созданы пустые списки задач.
  • Если количество недель для хранения выполненных задач указано нулевым, храниться будут задачи за одну неделю.
Аргументы
[in]_DataИсходные данные для работы класса.
Исключения
std::exception
  • Указанный файл содержит невалидную структуру.

AllTasks::String_t AllTasks::GetFullTaskText ( Id_t  _Id) const
private

Функция получения текста задачи с учетом текста родительской задачи.

  • Функция рекурсивно 'поднимается' вверх по иерархии задач для указанной, суммируя тексты задач, у которых установлен флаг m_IsAddParentTaskText, до тех пор, пока не наткнется на задачу, у которой этот флаг не установлен.
  • Текст родительской задачи добавляется слева от текста текущей задачи.
  • У текста родительской задачи удаляется последняя точка и добавляется ": ".
  • Если добавляется текст родительской задачи, у текущей первая буква приводится к нижнему регистру.
Аргументы
[in]_IdИдентификатор задачи, для которй будет искаться родительская задача.
Возвращает
Текст указанной задачи У указанной задачи не установлен флаг m_IsAddParentTaskText.
Исключения
std::exception
  • Задачи с указанным идентификатором не существует.
