Элементы перечислений

Задача может быть завершена/возвращена.

Функция изменения типа задачи.

  • Функция изменяет значение флагов для указанного типа задачи.
  • Само изменение флагов реализовано в специализации шаблона для каждого типа.
Параметры шаблона
TTypeТип задачи, состояние которого объект должен принять (Process или Complete).
Исключения
std::exception
  • Задан неподдерживаемый тип задачи.
