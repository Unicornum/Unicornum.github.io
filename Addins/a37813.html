Подробное описание

Класс входит в Testing
Класс команды добавления тестового проекта для проекта.

  • Проект тестирования будет добавлен проекту, выбранному в обозревателе решений.
  • Проект тестирования будет добавлен в ту же папку решения, в которой расположен выбранный проект.
  • Экспортирован проект тестирования будет из пути, который можно получить из настроек; имя параметра: 'SvnUrlTestProject' из раздела 'Test'
  • Команда будет доступна, если:

    В решении есть общий проект тестирования.

У выбранного проекта еще нет проекта тестирования.

Выбранный проект расположен в двойной папке, совпадающей с его именем.
