Класс входит в Support
Базовый класс всех объектов расширений для Visual Studio.

  • Класс-наследник должен содержать конструктор по умолчанию, передающий конструктору этого класса имя своего модуля.
  • Класс-наследник должен переопределить функцию Initialyze(), вызвать в ней функцию базового класса и подписаться на нужные ему события.
