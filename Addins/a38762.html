SolutionFolderHelper::Item_t * SolutionFolderHelper::GetItem ( Items_t *  _pItems,
const CString &  _Name 
) const
protected

Функция для поиска в списке элементов элемента с указанным именем.

  • Прямой поиск по имени не работает в VS2013+, поэтому ищем перебором.
  • В VS2008 проект не будет найден по новому имени сразу после переименования.
Аргументы
[in]_pItemsИсходный список элементов.
[in]_NameИмя искомого элемента.
Исключения
std::exception
  • Нет элемента с указанным именем.
  • Действие невозможно (подробнее см. описание исключения).
