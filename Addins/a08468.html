ClassInfo::ClassFiles_t ClassInfo::GetExistFiles ( void ) const

Список файлов, относящихся к классу.

  • Функция возвращает список полных путей к существующим файлам, содержащим объявление/реализацию класса, а также пути к файлу теста этого класса.
  • Проверяются файлы с именем, совпадающим с именем класса и расширениями: .hpp, .cpp, .inl.
