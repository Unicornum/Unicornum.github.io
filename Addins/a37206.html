◆ UpdateCopyrightYear()

void DocumentHelper::UpdateCopyrightYear ( void  )

Функция обновления года в строке copyright'a.

  • Должна вызываться, когда курсор стоит на строке объявления класса.
  • Если год в строке copyright'a совпадает с текущим, строка не будет изменена.
  • Если год не совпадает с текущим - он будет заменен на строку вида '2013 - 2015', где второй год - текущий.
  • Если строка copyright'a содержит годы в виде '2013 - 2014', второй год в этой записи будет заменен на текущий.
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).

◆ UpdateVersion()

void DocumentHelper::UpdateVersion ( int  _Index)

Функция увеличения версии в описании класса.

  • Добавляет строку версии в раздел version с увеличенным значением, зависящим от входного параметра.
  • Все значения версии правее указанного номера будут установлены в 0.
  • Должна вызываться, когда курсор находится ниже строки date.
Аргументы
[in]_IndexНомер версии, которая будет увеличена на 1 (1.2.3.4).
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).
  • _Index не попадает в диапазон [1...4].
