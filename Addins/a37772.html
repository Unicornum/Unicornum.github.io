void Replace::DoLocalProcess ( void  )
overrideprotectedvirtual

Функция обработки данных, переданных в конструкторе.

  • У переданных данных RegexString должна содержать регулярное выражение, ReplaceString - строку на замену, SourceStrings - список обрабатываемых строк (поле String у каждого элемента).
  • Функция обработает все строки из SourceStrings и заполнит поля ProcessingResults (будет только один элемент).
  • ProcessingResults[0].Result будет содержать строку результата замены или описание ошибки, если введено некорректное регулярное выражение.
  • ProcessingResults[0].IsInvalidRegex будет содержать true, если введено некорректное регулярное выражение.
  • Если список обрабатываемых строк пуст, в SourceStrings будет добавлен элемент, содержащий пустую строку для String, а в ProcessingResults будет добавлен один элемент, Result которого будет содержащать строку 'Нет исходных строк для обработки.'.

Замещает model::Common.
