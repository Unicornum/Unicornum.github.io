TestClassProject::TestClass_t TestClassProject::Create ( const CString &  _UrlToTestDirectory,
const SolutionFolder_t &  _Folder 
) const

Создание тестового проекта класса.

  • Экспортирует заготовку тестового проекта класса из репозитория svn в папку тестового проекта проекта.
  • Переименовывает файл экспортированного проекта.
  • Подключает добавленный проект в папку 'Test.Classes' папки проекта, которому принадлежит файл класса.
  • Меняет имя проекта на 'Test.CurrentClassName'.
  • Делает добавленный тестовый проект запускаемым.
Аргументы
[in]_UrlToTestDirectoryUrl к папке прототипов тестовых проектов в репозитории svn.
[in]_FolderРодительская папка проекта, которому принадлежит класс.
Возвращает
Объект созданного тестового проекта класса.
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).
