Addins

Решение представляет собой набор надстроек для IDE, предназначенных для:

  • автоматизации рутинных операций при работе с кодом;
  • упрощения работы со вспомогательными утилитами (прямо из среды разработки);

Поддерживаются следующие IDE:

  • Visual Studio.
