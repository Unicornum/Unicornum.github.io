Подробное описание

Класс входит в Version
Класс команды обновления номеров ревизий кода в файле версии решения.

  • Обновляется равизия файлов решения.
  • Путь к папке файла версии решения берется из настроек (параметр 'PathToVersionFile' в разделе Common).
  • Если в конструкторе не задано имя лога, версия будет выводиться в окно, имя которого будет запрошено у настроек, как параметр 'NameBuildWindow'.
