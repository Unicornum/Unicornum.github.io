void Common::DoProcess ( void  )
finaloverridevirtual

Функция обработки данных, переданных в конструкторе (общая логика).

  • У переданных данных RegexString должна содержать регулярное выражение, SourceStrings - список обрабатываемых строк (поле String у каждого элемента).
  • Функция обработает все строки из SourceStrings и заполнит поля ProcessingResults.
  • ProcessingResults[0].Result будет содержать:
    • описание ошибки, если введено некорректное регулярное выражение.
  • ProcessingResults[0].IsInvalidRegex будет содержать true, если введено некорректное регулярное выражение.
  • Если список обрабатываемых строк пуст, в SourceStrings будет добавлен элемент, содержащий пустую строку для String, а в ProcessingResults будет добавлен один элемент, Result которого будет содержащать строку 'Нет исходных строк для обработки.'.

Замещает model::IModel.
