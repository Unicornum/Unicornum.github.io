Подробное описание

Класс входит в ProjectTasks
Класс доступа к параметрам списка задач каждого конкретного проекта.

  • Параметры читаются из файла, указанного в конструкторе.
  • Если в указанном файле отсутствует раздел параметров, он будет создан.
  • Позволяет, например, хранить размеры окна программы для каждого проекта в отдельности.
