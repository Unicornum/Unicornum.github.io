virtual void model::ITasks::CreateChild ( size_t  _Id)
pure virtual

Функция добавления дочерней задачи для указанной.

  • Новая задача должна быть добавлена в конец списка дочерних задач указанной задачи.
  • Текущей задаче должен быть выставлен флаг IsExpanded.
Аргументы
[in]_IdИдентификатор задачи, в список дочерних которой будет добавлена задача.
Исключения
std::exception
  • Задачи с указанным идентификатором не существует.

Замещается в model::AllTasks.
