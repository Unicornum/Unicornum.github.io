::boost::signals2::signal<void(const wrappers::vs::Project &)> connection_points::IExtensions< TUsingExtensionData >::OnProjectAdded

Сигнал будет вызываться сразу после загрузки проекта.

Вызывается:

  • При создании нового проекта.
  • При подключении существующего проекта (вручную или надстройкой).
  • При загрузке проектов для папки решения.
  • VS2013+: для каждого проекта при загрузке решения.
