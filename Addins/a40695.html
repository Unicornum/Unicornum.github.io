Проект Main

Главный модуль подключения к Visual Studio: загружает и активирует все остальные модули при старте IDE.

Страница проекта: Main

Проект MainUI

Проект dll, содержащей ресурсы, необходимые для работы главного модуля VSPackage.

Страница проекта: MainUI

Проект Data

Проект содержит файлы, необходимые для работы надстроек (настройки, шаблоны документов и т.п.), которые копируются в папку построения при сборке проекта.

Страница проекта: Data

Проект Properties

Проект точки подключения к Visual Studio, реализующий логику подключения к проекту файла общей страницы свойств решения.

Страница проекта: Properties

Проект Version

Надстройка предназначена для установки .exe и .dll файлам версии решения, включая номер ревизии файлов решения.

Страница проекта: Version

Проект TimeTracker

Проект модуля учета времени работы над решением.

Страница проекта: TimeTracker

Проект Service

Проект модуля вспомогательных действий.

Страница проекта: Service

Проект UnicornumCodeAssistant

Проект модуля точки подключения к Visual Studio, обеспечивающий работу команд, доступных из меню.

Страница проекта: UnicornumCodeAssistant

Проект Support

Проект, содержащий вспомогательные классы, общие для проектов раздела.

Страница проекта: Support
