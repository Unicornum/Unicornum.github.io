Конструктор создания типа действия из индекса списка типов.

  • Предназанчен для создания объекта типа из индекса выбранного типа в выпадающем списке.
Аргументы
[in]_IndexИндекс типа, соответствующий списку, возвращаемому функцией GetTypeValues().
Исключения
std::exception
  • Индекс превышает количество типов, возвращаемое функцией GetTypeValues().

Функция увеличения номера версии в зависимости от текущего выбранного типа.
