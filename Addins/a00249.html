Подробное описание

Класс входит в Проект Documentation
Класс генерации подставного класса.

  • Генерирует подставной класс в расчете на использование библиотеки GoogleMock.
  • Подставной класс генерируется для класса, объявленного в заголовочном файле, путь к которому передается конструктору.
  • Генерирует подставной класс с простым набором подставных функций (что позволяет проверять вызовы функций для объектов, переданных как ссылки/указатели).
  • Генерирует подставной класс с использованием прокси-класса (что позволяет проверять вызовы конструкторов и статических функций).
  • Подставной класс генерируется в файле, совпадающем с именем исходного заголовочного файла с добавлением '.mock' между именем файла и расширением.
  • Созданный подставной класс будет иметь то же имя, что и исходный и будет расположен в том же пространстве имен, но 'обернут' в пространство имен mock.
  • Созданный подставной класс будет содержать все using объявления и открытые конструкторы/функции исходного класса.
  • Созданный файл подставного класса добавляется в общий заголовочный файл подставных классов текущей папки (файл с именем "<имя папки>.mock.hpp" в родительской папке текущей; если такого файла нет, он создается).
