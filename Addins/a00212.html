Подробное описание

Класс входит в Проект Macros
Класс команды вставки заготовки текста.

  • Заготовка текста '_T("")' вставляется в место расположения курсора.
  • Курсор после вставки будет располагаться внутри кавычек.

Функция получения текста, отображаемого в строке команды.

  • Каждый класс-наследник должен переопределить функцию и определить текст, отображаемый для текущей команды.
  • По умолчанию возвращает имя класса.
