CString CppCheckSolutionQuickProcess::GetCaption ( void  ) const
virtual

Функция получения текста, отображаемого в строке команды.

  • Каждый класс-наследник должен переопределить функцию и определить текст, отображаемый для текущей команды.
  • По умолчанию возвращает имя класса.

Переопределяет метод предка useful_commands::IMenuCommand.

int CppCheckSolutionQuickProcess::GetPictureId ( void  ) const
virtual

Функция получения идентификатора иконки команды.

  • Каждый класс-наследник должен переопределить функцию и определить иконку, отображаемую для текущей команды (по умолчанию - без иконки).
  • Положительное значение: номер системной иконки.
  • Отрицательное значение: номер иконки в файле '.\1049\icons.dll' (путь относительно папки надстройки).

Переопределяет метод предка useful_commands::vs::Unknown.

bool CppCheckSolutionQuickProcess::IsEnabled ( void  ) const
virtual

Функция проверки доступности команды.

  • По умолчанию возвращает false.
Возвращает
true Команда доступна (может быть выполнена).
false Команда недоступна (будет отображаться в меню, но не может быть выполнена).
