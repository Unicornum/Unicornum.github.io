Класс входит в Document
Класс-обертка для EnvDTE::TextSelection.

  • Предназначен для работы с одной строкой документа.
  • При вызове константных функций курсор останется в той же строке документа.

Пример использования:

Удаление строки, на которой стоит курсор:

GetDTE().Documents[Active].Lines[Current] = _T("");

Пример использования:

Проход по строкам, которые содержат "nullptr":

using namespace wrappers::vs;
Document Example = GetDTE().Documents[Active];
selection::Line FindedLine = Example.Lines[-1];
while (true)
{
FindedLine = Example.Lines[FindedLine.GetIndex() + 1].Find(_T("nullptr"));
if (FindedLine.GetIndex() < 0) break;
// FindedLine содержит "nullptr"
long Index = FindedLine.GetIndex();
}

◆ Find()

Line wrappers::vs::selection::Line::Find ( const CString & _Pattern,
Options::Value _Options = Options::Forward ) const

Поиск текста.

  • Функция производит поиск в документе указанного текста относительно текущей строки.
  • Функция выделяет найденный текст.
  • Независимо от направления поиска, он начинается с текущей строки.
Аргументы
[in]_PatternСтрока-образец для поиска.
[in]_OptionsПараметры поиска.
Возвращает
Line Найденная строка (ее индекс будет равен -1, если искомая строка отсутствует).
Исключения
std::exception
  • Ошибка при поиске строки.

Line & wrappers::vs::selection::Line::operator= ( const CString & _Text)

Запись текста в строку.

  • При записи новой строки нужно добавлять в ее конец символы переноса строки.
  • Для вставки новой строки после текущей, следует скопировать текущую строку, добавить к ней текст новой строки и записать результат в текущую строку.
Аргументы

Возвращает
Line Объект текущей строки.
Исключения
std::exception
