CString IMenuCommand::GetCaption ( void  ) const
virtual

Функция получения текста, отображаемого в строке команды.

  • Каждый класс-наследник должен переопределить функцию и определить текст, отображаемый для текущей команды.
  • По умолчанию возвращает имя класса.

Переопределяется в useful_commands::vs::testing::AddProjectTest, useful_commands::vs::testing::AddClassTest, core::useful_commands::vs::testing::AddSolutionTest, core::useful_commands::vs::utilities::OpenRegExWindow, core::useful_commands::vs::utilities::OpenSettings, useful_commands::vs::doxygen::IncreaseVersionCommon, useful_commands::vs::macros::InsertPragmaMessage, useful_commands::vs::doxygen::AddDescription, useful_commands::vs::macros::CommentLineBreak, useful_commands::vs::macros::InsertTextBlank, useful_commands::vs::utilities::AStyleProcess, useful_commands::vs::testing::AddTemplateTest, useful_commands::vs::utilities::CppCheckSolutionQuickProcess, useful_commands::vs::utilities::AddSelectedTextToUsertypeFile, useful_commands::vs::common::OpenDocAddins, useful_commands::vs::Test, useful_commands::vs::doxygen::OpenDoxygenDoc, useful_commands::vs::doxygen::OpenOwnLibsDoc, useful_commands::vs::doxygen::OpenSolutionDoc и useful_commands::vs::common::VersionAddins.
