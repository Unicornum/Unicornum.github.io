Примеры использования:

// Создаем объект, который будет содержать настройки 'UniCA':
Settings oSettings(_T("UniCA"));
// Загружаем настройки для Visual Studio 2008 из файла 'Settings.xml':
oSettings.Load(_T("VS2008"), _T("Settings.xml"));
// Получаем значение параметра 'OwnLibs':
CString PathToOwnLibs = oSettings[Settings::Section::Common][_T("OwnLibs"))];
// Создаем объект, который будет содержать настройки 'UniCA':
Settings oSettings(_T("UniCA"));
// Загружаем настройки для Visual Studio 2008 из файла 'Settings.xml':
oSettings.Load(_T("VS2008"), _T("Settings.xml"));
// Получаем объект, содержащий параметры раздела Test:
Settings::Section TestSettings = oSettings[Settings::Section::Test];
// Получаем значение параметра 'PathToTestAll':
CString PathToProjectTestAll = TestSettings[_T("PathToTestAll")];

Конструктор(ы)

Settings::Settings ( const CString &  _AddinName = _T("Unknown"))
explicit

Конструктор класса.

  • Указанное имя надстройки будет доступно как параметр раздела Common с именем 'HostName'.
  • В debug версии программы к имени надстройки будет добавлено "debug".
  • Будет сформирован параметр раздела Common с именем 'HostPath', представляющий собой путь к модулю текущей надстройки.
Аргументы
[in]_AddinNameИмя надстройки.

void Settings::Load ( const CString &  _VersionIDE,
const CString &  _FileName = _T("Settings.xml") 
)

Функция для загрузки параметров из файла.

  • Строка версии IDE должна совпадать с именем раздела файла настроек, из которого будут читаться значения параметров.
Аргументы
[in]_VersionIDEСтрока версии IDE, для которой будут загружены параметры.
[in]_FileNameИмя файла, из которого будут загружены параметры.
Исключения
std::exception
  • Не найден указанный файл.
  • Нет обязательного раздела.
