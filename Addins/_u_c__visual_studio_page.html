Проекты раздела

Проект StartUp

Проект классов команд, выполняемых при запуске надстройки:

  • Вывод в лог информации о запускаемой надстройке.
  • Загрузка параметров надстройки.
  • Удаление команд, оставшихся от предыдущего запуска.
  • Копирование файла usertype.dat.
