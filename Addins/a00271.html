void Prefatory::DeleteAutoInsertSection ( const Path_t &  _PathToFile)
private

Функция удаления вставленного при предыдущей обработке текста.

  • Будет вызвана для всех .hpp файлов, расположенных в папках Documentation.
  • Удаляет в указанном файле весь текст между тегами '\begin_auto_insert_section' и '\end_auto_insert_section', включая пустую строку после '\end_auto_insert_section' (если такая есть).
  • Если найден открывающий тег, но не найден закрывающий, в лог будет добавлена ошибка.
  • Не закрытая секция не удаляется.
  • Если указанного файла нет или его невозможно открыть, в лог будет добавлена ошибка.
Аргументы
[in]_PathToFileПуть к файлу, который должен быть обработан.

void Prefatory::DoProcess ( const Path_t &  _Path)
private

Функция предварительной обработки документации.

  • Обрабатывает файлы в указанной папке и ее подпапках.
  • Обрабатываются файлы Page.hpp, содержащиеся в папках с именем 'Documentation' и '(Documentation)', при этом папки 'Documentation' считаются устаревшими и для них выводится предупреждение в лог.
  • Игнорируются папки, вложенные в папки с именами 'html', '.svn', 'Test.'.
  • Собирает воедино всю логику предварительной обработки документации.
Аргументы
[in]_PathПуть к папке, файлы которого должны быть обработаны.
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).

Функция для обработки файла.

  • Читает текстовый файл по строке и передает ее заданному в качестве параметра функтору.
  • Оператор '()' функтора будет получать в качестве параметров путь к обрабатываемому файлу, текущую обрабатываемую строку и номер (не индекс!) текущей строки.
Аргументы
[in]_PathToFileПуть к файлу, который будет обрабатываться.
[in]_FunctorОбъект функтора, который будет производить обработку.
Возвращает
true Обработка файла прошла до конца.
false Не удалось открыть указанный файл.

void Prefatory::FindInsertedSection ( const Path_t &  _PathToFile)
private

Функция поиска текста для найденных тегов.

  • Заполняет в элементах m_SectionInfos для найденного SectionName поле InsertedText.
  • Проверяет, что SectionName - это имя тега секции (в противном случае записывает ошибку в лог).
  • Извлекается текст от найденной секции до следующей пустой строки (содержащей тольк звездочку и пробелы), секции, подсекции или завершающих символов комментария (звездочка - слеш).
Аргументы
[in]_PathToFileПуть к файлу, который должен быть обработан.

void Prefatory::FindInsertSection ( const Path_t &  _PathToFile)
private

Функция поиска мест, в которые будут добавляться описания.

  • Будет вызвана для всех .hpp файлов, расположенных в папках Documentation.
  • Вызывается после очистки файла от устаревшего содержимого.
  • Собирает список строк файлов, содержащих тег '\insert_SectionName'.
Аргументы
[in]_PathToFileПуть к файлу, который должен быть обработан.

Функция для вставки описаний.

  • Расставляет собранную в m_SectionInfos информацию 'по местам'.
