Функции

template<class T >
Proxy<T>::Ptr_t wrappers::helper::Is ( const T &  _Item)

Функция - фасад для однотипной работы с реализациями разных интерфейсов.

  • Для реализации конкретного интерфейса нужно: # Определить интерфейс с нужными функциями
    class ISolution
    {
    public:
    virtual bool ContainsTestAll(void) = 0;
    };

    Создать реализацию этого интерфейса, получающую в конструкторе объект

исходного класса.

class SolutionImpl : public ISolution
{
public:
virtual bool ContainsTestAll(void) {...}
public:
SolutionImpl(const Solution &) {...}
};

Создать специализацию прокси-класса, связывающую между собой исходный

класс, интерфейс и реализацию.

template<>
class Proxy<Solution> : public ImplPtr<ISolution>
{
public:
Proxy(const Solution & _Item) :
ImplPtr(new SolutionImpl(_Item))
{
}
};

# Пользоваться так:

Solution Sln;
if (Is(Sln)->ContainsTestAll()) {...}
Аргументы
[in]_ItemОбъект исходного класса, для которого будет производиться проверка.
