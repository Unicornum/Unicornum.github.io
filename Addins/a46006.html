Utilities

Введение

Раздел вспомогательных классов, которые используются в других разделах решения.

Проекты раздела

Проект Dialog

Проект, содержащий классы диалоговых окон:

  • Для отображения прогресса некоего процесса (в процентах и просто 'бегущей' строкой).

Страница проекта: Dialog

Проект EnvironmentVariables

Проект классов для установки значений переменных окружения для программ:

  • Doxygen.

Страница проекта: EnvironmentVariables

Проект Execute

Проект содержит классы-фасады для запуска внешних программ (утилит), обеспечивающих работу соответствующих команд надстройки.

Страница проекта: Execute

Проект Filter

Проект классов фильтрации полученного от выполняемой программы консольного вывода.

Страница проекта: Filter

Проект Logger

Проект классов, обеспечивающих вывод информации:

  • В выводное окно Visual Studio.

Страница проекта: Logger

Проект Utilities

Проект содержит отдельные вспомогательные классы общего характера, используемые в других разделах решения (проект Utilities_x64 включает те же файлы, но собирается как x64 для проектов, которым это нужно - см., например DevelopmentInfo).

Страница проекта: Utilities
