Подробное описание

Класс входит в Проект UsefulCommands
Класс команды вставки сообщения #pragma chMSG().

  • Вставляет в текущую позицию курсора заготовку сообщения #pragma chMSG(), которое при компиляции попадет в выводное окно Visual Studio, а при построении документации окажется в списке TODO.
  • Сообщение вставляется в текущую строку.
  • 'Горячая' клавиша команды - "ALT + M'.

CString InsertPragmaMessage::GetCaption ( void  ) const
virtual

Функция получения текста, отображаемого в строке команды.

  • Каждый класс-наследник должен переопределить функцию и определить текст, отображаемый для текущей команды.
  • По умолчанию возвращает имя класса.

Переопределяет метод предка useful_commands::IMenuCommand.

InsertPragmaMessage::HotKey_t InsertPragmaMessage::GetHotKey ( void  ) const
virtual

Функция получения 'горячей' клавиши для команды.

  • По умолчанию 'горячая' клавиша отсутствует.
  • Каждый класс-наследник может переопределить функцию и определить 'горячую' клавишу, при нажатии которой будет выполнена эта команда.
  • Способ задания 'горячей' клавиши см. у класса utilities::HotKey.

Переопределяет метод предка useful_commands::vs::Unknown.
