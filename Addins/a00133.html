size_t core::utilities::Calendar::GetWeekNumber ( void  ) const

Функция вычисления номера недели для даты, переданной конструктору.

  • Если в качестве первого дня недели конструктору передано 1 (понедельник), то первой неделей будет считаться 1...7 января 1900 года, если 2 (вторник), то 2...8 января 1900 года и т.д.
