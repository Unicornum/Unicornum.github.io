template<class TDialogProgress, class TFilter = filter::Empty>
class utilities::progress::ConsoleProgram< TDialogProgress, TFilter >

Класс входит в Проект Progress
Класс-обертка для отображения прогресса процесса.

  • Обрабатываемый процесс передается в конструкторе.
  • Обеспечивает основную логику обработки результатов выполнения программы.
Параметры шаблона
TDialogProgressКласс диалогового окна, которое будет использоваться для отображания прогресса процесса.
Должен:
  • В конструкторе получать указатель на utilities::progress::IProgress и заголовок окна.
  • Cодержать функцию int ShowModal(void); , которая отображает модальное диалоговое окно и завершается при закрытии этого окна, возвращая код, зависящий от способа закрытия окна (IDOK, IDCANCEL) или -1, если окно создать не удалось.
TFilterКласс фильтра, который будет использоваться для фильтрации текста, получаемого от работающей программы; должен быть функтором, т.е. должен содержать оператор вида
CString operator() (const CString & _Text);
, который будет вызываться для каждой строки получаемой информации.
  • Объект фильтра создается один раз вместе с объектом класса, поэтому его можно использовать (например) для исключения повторяющихся строк.
  • Возвращаемый оперотором результат будет записан в лог работы, переносы строк не нужны, к не пустым строкам они будут добавлены автоматически.
