Класс входит в RegexTester
Базовый класс панелей главного tabctrl'a окна программы.

Параметры шаблона
TPanelКласс панели, реализацию которой должен обеспечить текущий класс.
Должна содержать:
  • элемент строки реулярного выражения (m_pRegexString),
  • tabctrl (m_pNotebookSourceResults), содержащий
    • элемент исходных строк для обработки (m_pSourceStrings),
    • элемент отображения результата обработки (m_pProcessedResult).
  • обработчики событий
    • активации панели,
    • изменения регулярного выражения,
    • изменения обрабатываемых строк.
