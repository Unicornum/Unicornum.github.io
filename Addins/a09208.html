◆ Add()

Command Commands::Add ( const AddIn & _AddIn,
const CString & _Name,
const CString & _Caption,
LONG _IDPicture ) const

Создать команду.

  • Если команда уже существует, функция вернет объект этой команды.
  • Функция создает новую команду с указанным именем _Name, т.е. Addins.Connect._Name.
Аргументы
[in]_AddInЭкземпляр объекта надстройки.
[in]_NameИмя создаваемой команды.
[in]_CaptionТекст, отображаемый на строке команды, если она находится в меню.
[in]_IDPictureИдентификатор иконки команды (положительное число, если это номер иконки самой Студии и отрицательное, если номер иконки из внешней DLL).
Исключения
std::exception
Создать команду невозможно.
