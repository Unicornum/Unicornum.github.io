Класс входит в EnvironmentVariables
Базовый класс классов установки значений переменных окружения.

  • Класс-наследник (который должен создаваться для конкретной программы) должен передать конструктору базового класса список требуемых программе переменных окружения.
  • Класс-наследник должен получать в конструкторе 'const Param_t &'.
Параметры шаблона
TParamТип параметра, который будет получать в конструкторе класс-наследник.
