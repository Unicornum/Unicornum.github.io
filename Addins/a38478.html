Подробное описание

Класс входит в Utilities
Класс статистики о работе над решением.

  • Загружает из файла, указанного в конструкторе информацию о текущем решении, позволяет изменить ее и сохранить результат в тот же файл.

Конструктор класса.

  • Загружает информацию о текущем решении из указанного файла.
  • Если файла не существует, текущая информация будет содержать 'нулевые' значения.
Аргументы
[in]_PathToSourceFileПуть к файлу, из которого будет загружена информация.
[in]_SavePeriodПериод времени, через который будет производиться сохранение текущего состояния в файл.
[in]_IgnoreTimeЕсли промежуток времени между вызовами UpdateTime() будет больше указанного, он будет проигнорирован.
Исключения
std::exception
  • Невозможно открыть указанный файл.
  • Указанный файл имеет некорректную структуру.

void SolutionDevelopmentInfo::Save ( void  ) const
private

Функция сохранения изменений в файл.

  • Текущее состояние будет сохранено в файл, указанный в конструкторе.
  • Будет сгенерирован файл (с тем же путем/именем, что указан в конструкторе, но с расширением .hpp), содержащий страницу статистики работы над решением для включения в документацию Doxygen (метка для ссылок - SolutionStatisticsPage).
  • Сгенерированный файл для документации перезапишет существующий.
Исключения
std::exception
  • Не удалось открыть/создать один из файлов.
