Unicornum Code Assistant

Модуль добавляет разделы 'UniCA' в различные меню Visual Studio.

Unicornum Code Assistant

Properties

Модуль обеспечивает автоматическое подключение специализированной страницы свойств ко всем конфигурациям всех проектов решения.

Используемые параметры настроек

Описания параметров см. в утилите редактирования настроек.

  • VisualStudio -> PathToSolutionPropsFile.
Предупреждения
Из-за того, что Visual Studio генерирует событие удаления/выгрузки проекта из решения уже после того, как проект фактически удален (и его модификация уже невозможна) не удалось реализовать автоматическое отключение специализированной страницы свойств при удалении проекта. В результате, если необходимо перенести проект в другую папку, после его удаления из решения необходимо открыть файл проекта в текстовом редакторе и вручную удалить подключенную страницу свойств.
Необходимо сделать
Проверить, что событие удаления проекта все еще генерируется после удаления.
Необходимо сделать
Поскольку в Vs2017 проект с несуществующей страницей свойств загружается успешно, реализовать удаление таких страниц свойств?

Service

Модуль обеспечивает выполнение следующих функций:

  • Звуковой сигнал при обнаружении Visual Studio утечки памяти при завершении работы программы.
  • Подсчет времени сборки проектов с выводом отчета с упорядочиванием по убыванию оного (для настройки см. параметры в разделе VisualStudio -> ProjectBuildStatistic).

Time Tracker

Модуль предназначен для автоматизация учета времени, потраченного на разработку ПО и простого доступа к результатам этого учета:

  • Затраченное время учитывается по действиям пользователя во время работы с Visual Studio (изменение текста открытого документа, выбор другого элемента в обозревателе решений, переключение активного окна).
  • Генерируется специальная страница с отчетом о времени работы над текущим решением для включения в документацию решения.
  • Команда Статистика о работе над решением в меню решения запускает специальную утилиту, которая отображает статистику работы над кодом за последние две недели.

Используемые параметры настроек

Описания параметров см. в утилите редактирования настроек.

  • Utilities -> BeginWeekDayNumber.
  • Utilities -> ProjectTasks -> все параметры раздела.

Version

Модуль предназначен для автоматизации работы с версией решения и выполняет следующие действия:

  • Перед построением проекта обновляет номера ревизий и даты/времени файлов решения файле версии решения.
  • Перед построением .exe или .dll проекта генерирует для них файл version.rc, содержащий ссылку на общий файл version.rc (это для того, чтобы собранные бинарники содержали текущую версию решения) и подключает его к проекту.

Используемые параметры настроек
