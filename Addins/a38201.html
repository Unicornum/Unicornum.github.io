◆ Check()

virtual CString core::utilities::filter::IFilter::Check ( const CString &  _Text)
pure virtual

Функция для обработки строк, получаемых от запущенной программы.

  • Во время работы программы функция будет вызываться для каждой отдельной строки консольного вывода запущенной программы.
  • Возвращенная строка будет записана в лог работы программы.
Аргументы
[in]_TextОчередная строка.
Возвращает
_Text Строка несет полезную информацию и должна быть записана в лог обработки.
_T("") Строка должна быть проигнорирована.

Замещается в core::utilities::filter::CppCheck, core::utilities::filter::Empty и core::utilities::filter::Doxygen.

◆ GetProgress()

virtual int core::utilities::filter::IFilter::GetProgress ( void  ) const
pure virtual

Функция для получения текущего прогресса обработки.

  • Значение прогресса обработки должно вычисляться в функции Check() при обработке получаемых строк.
  • Функция будет использоваться только в при использовании диалогового окна, отображающего прогресс обработки в процентах (для окна, отображающего 'бегущую строку' эта функция может просто возвращать 0).
Возвращает
Значение текущего прогресса обработки в процентах.

Замещается в core::utilities::filter::CppCheck, core::utilities::filter::Empty и core::utilities::filter::Doxygen.
