Перечисления

◆ AllowedFlags

enum support::DataTask::AllowedFlags : size_t

Флаги для обозначения допустимых действий над задачей.

Элементы перечислений
Editable 

Текст задачи может быть изменен.

CreateChild 

Задаче можно добавлять дочерние подзадачи.

Completed 

Задача может быть завершена/возвращена.

BlockCompleted 

Задача заблокирована для выполнения, например, это может быть родительская задача, у которй есть не завершенные подзадачи.
