Подробное описание

Класс входит в Documentation
Класс команды обработки файла результата построения решения скриптом.

  • Путь к файлу результата обработки берется из настроек - как параметр 'PathToMSBuildResult' из раздела Documentation.
  • Предполагается, что этот файл содержит только error'ы и warning'и.
  • Пустой и не существующий файлы будут проигнорированы.
