Конструктор класса.

После создания объекта m_Info будет содержать:

  • Для рабочей копии:
    Path: P:\projects\Addins_vs2013
    Working Copy Root Path: P:\projects\Addins_vs2013
    URL: https://svn/svn/projects/tools/Addins/branches/CheckPoints/2015.08.02
    Relative URL: ^/tools/Addins/branches/CheckPoints/2015.08.02
    Repository Root: https://svn/svn/projects
    Repository UUID: 4951e77f-dbfd-5242-9449-5adf9c224220
    Revision: 2091
    Node Kind: directory
    Schedule: normal
    Last Changed Author: CTAPOBEP
    Last Changed Rev: 2090
    Last Changed Date: 2015-08-06 23:11:57 +0300 (Чт, 06 авг 2015)
  • Для неверсированных файлов:
    svn: E155007: 'path' is not a working copy
  • Для рабочей копии, с репозиторием которой нет связи
    svn: E170013: Unable to connect to a repository at URL 'https://svn/svn/Libs/trunk'
    svn: E730065: Error running context: Сделана попытка выполнить операцию на сокете для недоступного хоста.
Аргументы
[in]_PathToWorkCopyПолный путь к папке рабочей копии, для которой будет извлекаться информация.
Исключения
std::exception
  • Не установлена программа TortoiseSVN.
