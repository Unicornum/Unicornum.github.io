Подробное описание

Класс входит в Проект Common
Базовый класс команд, которые будут выполняться в VisualStudio при выборе строки меню.

  • Классы-наследники должны переопределить функцию Execute(), в которой должно выполняться то действие, для которого этот класс создавался.
  • Классы-наследники должны переопределить функцию IsEnabled(), если требуется проверка доступности команды.
  • Классы-наследники должны переопределить функцию GetCaption(), которая должна возвращать отображаемую в меню строку команды.
  • Классы-наследники должны переопределить функцию GetPictureId(), если команде требуется специфическая иконка (идентификаторы иконок см. в файле "Core\UsefulCommands\Common\Common\Icons.hpp").
  • Классы-наследники должны переопределить функцию GetHotKey(), если требуется назначить команде 'горячую' клавишу.

MenuUnknown::HotKey_t core::useful_commands::vs::common::MenuUnknown::GetHotKey ( void  ) const
inlinevirtual

Функция получения 'горячей' клавиши для команды.

  • По умолчанию 'горячая' клавиша отсутствует.
  • Каждый класс-наследник может переопределить функцию и определить 'горячую' клавишу, при нажатии которой будет выполнена эта команда.
  • Способ задания 'горячей' клавиши см. у класса utilities::HotKey.

Переопределяется в useful_commands::vs::macros::InsertPragmaMessage, useful_commands::vs::macros::CommentLineBreak и useful_commands::vs::macros::InsertTextBlank.

int core::useful_commands::vs::common::MenuUnknown::GetPictureId ( void  ) const
inlinevirtual

Функция получения идентификатора иконки команды.

  • Каждый класс-наследник должен переопределить функцию и определить иконку, отображаемую для текущей команды (по умолчанию - без иконки).
  • Положительное значение: номер системной иконки.
  • Отрицательное значение: номер иконки в файле '.\1049\icons.dll' (путь относительно папки надстройки).

Переопределяется в useful_commands::vs::testing::AddClassTest, useful_commands::vs::utilities::CppCheckSolutionQuickProcess и useful_commands::vs::Test.

Функция получения статуса команды.
