inlinevirtual

Функция получения текста, отображаемого в строке команды.

  • Каждый класс-наследник должен переопределить функцию и определить текст, отображаемый для текущей команды.
  • По умолчанию возвращает имя класса.

Переопределяет метод предка useful_commands::IMenuCommand.
