Подробное описание

Класс входит в ProjectTasks
Класс формирования общего списка текущих задач всех проектов решения.

  • Класс формирует страницу со списком всех задач решения в формате документации Doxygen (имя страницы для ссылок - SolutionTasksPage).
  • Если такого файла нет, он будет создан.
  • При создании объекта класса он удаляет все записи, относящиеся к текущему проекту, при разрушении объекта - создает раздел текущего проекта с добавленными записями.
  • Раздел задач текущего проекта производится в конец страницы всех задач решения.
