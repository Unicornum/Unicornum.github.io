◆ Replace()

void wrappers::vs::Document::Replace ( const CString &  _Pattern,
const CString &  _Value 
)

Замена текста во всем документе.

  • Замена текста производится по всему документу без учете регистра символов.
  • Если при вызове функции было найдено и заменено несколько образцов текста, команда "Отмена" отменит не все, а только одно, последнее изменение.
Аргументы
[in]_PatternОбразец текста для поиска.
[in]_ValueТекст на замену.
Исключения
std::exception
  • Ошибка при замене текста.
