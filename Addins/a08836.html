◆ ToLog()

void core::utilities::vs::Report::ToLog ( size_t _Flag,
const CString & _Text )

Вывод сообщения в окно вывода.

  • Сообщение будет выведено в окно, имя которого указно в конструкторе.
  • Флаг Clear обрабатывается в первую очередь и очищает окно вывода; может использоваться вместе с любыми другими флагами.
  • Если установлен флаг Title, то текст будет выведен как заголовок (без указания времени и типа сообщения, а также дополнен символами '=' до длинны MaxMessageLenght), флаги [Fatal ... Debug] игнорируются.
  • Если установлен флаг Status, то текст будет выведен в статус-бар Visual Studio.
  • Если установлен один из флагов [Fatal ... Debug], будет выведено текущее время, тип сообщения (определяется по флагу) и само сообщение.
  • Сообщение с флагом Debug выводится только в отладочной версии программы.
  • Сообщение Simple будет выведено без времени и типа, просто указанная строка.
  • При выводе сообщений с типом Fatal, Error, Warning или установленном флаге Open будет окрыто выводное окно Visual Studio.
