Подробное описание

Класс входит в Macros
Обобщенный класс команды вставки заготовки текста.

  • Заготовка текста, указанная в конструкторе, вставляется в место расположения курсора.
  • Курсор после вставки будет сдвинут на указанное число позиций влево относительно последнего символа вставленного текста.

Функция получения текста, отображаемого в строке команды.

  • Каждый класс-наследник должен переопределить функцию и определить текст, отображаемый для текущей команды.
  • По умолчанию возвращает имя класса.

Переопределяет метод предка core::useful_commands::common::IMenuCommand.
