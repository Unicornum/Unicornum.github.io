Подробное описание

Класс входит в Utilities
Класс манипуляцией версией решения.

  • Если в указанной в конструкторе папке нет файла версии решения, он будет создан со значениями по умолчанию.
  • Отсутствующие в файле версии решения значения будут установлены в значения по умолчанию.
  • Генерируемый файл версии решения будет содержать макросы (в стиле С++) со значениями текущей версии решения.
  • Строковый формат версии: 'Version.Subversion.Subsubversion.Revision Architecture: x86.xx Build ЧЧ:ММ ДД.ММ.ГГГГ'

◆ Save()

void Version::Save ( void  ) const

Сохранение версии в файл версии решения.

  • В файл версии решения будут записаны текущие время/дата.
  • Строка copyright'a будет содержать текущий год, если разработка начата в этом году и два года через дефис (год начала разработки и текущий), в противном случае.
Исключения
std::exception
  • Не удалось открыть/создать файл версии решения.
