Класс входит в Patterns
Макрос для регистрации класса в фабрике объектов.

  • При использовании этого макроса требуется унаследовать интерфейс от Registrator<>.
  • Макрос ОБЯЗАТЕЛЬНО должен быть в заголовочном файле.
  • Если регистрируемый класс находится в пространстве имен, макрос также должен располагаться внутри этого пространства имен, а в качестве Class должно быть имя класса без его пространства имен.
Аргументы
[in]ClassРегистрируемый класс.
[in]ClassIdИдентификатор, по которому будут создаваться объекты класс (его следует передавать в функцию Factory<>::Create()).

Класс входит в Patterns
Макрос для регистрации класса в фабрике объектов по строковому имени.

  • При использовании этого макроса не требуется наследовать интерфейс от Registrator<>.
  • Класс будет зарегистрирован под строковым именем, совпадающим с именем класса.
  • Макрос ОБЯЗАТЕЛЬНО должен быть в заголовочном файле.
  • Если регистрируемый класс находится в пространстве имен, макрос также должен располагаться внутри этого пространства имен, а в качестве Class должно быть имя класса без его пространства имен.
Аргументы
[in]ClassРегистрируемый класс (регистрируется под собственным именем).
[in]InterfaceИнтерфейс, в фабрике которого регистрируется класс.
