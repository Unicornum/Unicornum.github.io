Класс входит в Проект System
Класс получения информации о текущем модуле.

  • Под 'текущим модулем' подразумевается exe или dll файл, внутри кода которого вызывается функция.

CurrentModule::Path_t CurrentModule::GetAppDataPath ( void  )
static

Функция получения пути к папке, содержащей данные программы для текущего пользователя.

  • Будет сформировано из пути к папке данных программ текущего пользователя, названий компании и продукта.
  • Если такого пути не существует, он будет создан.
  • Названия компании и продукта будут извлечены из версии текущего модуля (будут совпадать с макросами _ _ COMPANY_NAME _ _ и _ _ PRODUCT_NAME _ _ текущего решения), поэтому у программы, желающей использовать эту возможность должен быть подключен файл с ресурсом версии программы.
  • Если у текущего модуля нет ресурса версии, функция вернет путь к папке расположения модуля.
