Класс входит в проект Std
Класс создания объектов из пула памяти.

  • Создание объектов из пула памяти не приводит к фрагментации кучи.
  • Создание объектов из пула памяти происходит примерно в 3 раза быстрее, чем операторм new.
Параметры шаблона
TТип объектов, которые будут создаваться пулом.
TSizeРазмер пула памяти, который будет выделяться за раз (при создании первого объекта будет запрошена память в объеме TSize * sizeof(T), в дальнейшем по мере использования этого объема, память будет выделяться блоками такого же размера).
