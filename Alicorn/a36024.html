Класс входит в проект Document
Интерфейсный класс классов реализаций работы с документами.

  • Предоставляет интерфейс для классов, которые могут использоваться 'под капотом' класса alicorn::source::document::Universal.
  • Класс реализации должен иметь конструктор, принимающий в качестве параметра объект std::shared_ptr<alicorn::source::details::ISource>, который предоставляет доступ к содержимому файла.
