Подробное описание

Раздел классов локализации программы.

Локализация программы

Модуль локализации программы используется следующим образом:

  • В решении создать проект статической библиотеки Localization, в который поместить файлы, содержащие строки для различных языков (English.xml, Russian.xml и т.п.).
Заметки
Обязательно должен присуствовать файл Default.xml, который должен задавать иерархию строк и на основе которого будет генерироваться файл, содержащий идентификаторы используемых строк.

Пример файла Default.xml:

<?xml version="1.0" encoding="utf-8"?>
<main>
<path>Languages</path>
<Text1>Example text 1</Text1>
<Text2>Example text 2</Text2>
<child>
<Text3>Example text 3</Text3>
</child>
</main>
  • Добавить в решение проект утилиты Externals\Tools\Localization.
  • В качестве события после сборки проекту статической библиотеки Localization добавить запуск утилиты Localization, передав ей качестве параметра путь к файлу Default.xml (эта утилита сгенерирует в той же папке файл Localization.hpp, содержащий идентификаторы строк и вспомогательный класс Localized).

Пример сгенерированного файла:

#pragma once
#include <alicorn\localization.hpp>
// !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!! ВАЖНО !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
// Это АВТОМАТИЧЕСКИ сгенерированный файл, все изменения будут перезаписаны
// при сборке проекта Localization.
// !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!! ВАЖНО !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
namespace main
{
enum : size_t
{
Text1 = 14155219958802156194,
Text2 = 2627504610787872273,
};
namespace child
{
enum : size_t
{
Text3 = 8777275511942650506,
};
} // namespace child
} // namespace main
namespace alicorn
{
namespace modules
{
namespace localization
{
class Localized final
{
public:
static auto Get(size_t _StringId) { return GetInstance().Get(_StringId); }
static void Reload(void) { GetInstance().Load(); }
private:
static Localization & GetInstance(void)
{
return Localization::GetInstance(L"Languages");
}
};
} // namespace localization
} // namespace modules
} // namespace alicorn

  • В качестве события после сборки проекту статической библиотеки Localization добавить копирование xml файлов со строками в папку построения программы (с учетом относительной папки расположения этих файлов, заданной как раздел path).
  • В нужном месте программы включить файл Localization.hpp и использовать для получения строки выбранного в настройках языка записи вида
#include <Localization\Localization.hpp>
const auto Text1 = Localized::Get(::main::Text1); // "Example text 1"
const auto Text2 = Localized::Get(::main::Text2); // "Example text 2"
const auto Text3 = Localized::Get(::main::child::Text3); // "Example text 3"
  • В настройках в качестве выбранного языка программы следует указывать (в качестве значения параметра Language раздела Localization) имя файла .xml, содержащего строки нужного языка ("English", "Russian" и т.п.).
