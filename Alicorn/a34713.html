Подробное описание

Класс входит в Std
Класс-обертка для классов конкретных кодировок.

Класс предназначен для:

  • Установки локали по умолчанию (глобальной или для текущей области видимости).
  • Указания используемой кодовой страницы при преобразовании строк при помощи функции string_cast<>().
Заметки
Установка локали для текущей области видимости не означает, что локаль устанавливается только для этой функции! Это значит, что устанавливается глобальная локаль, которая будет восстановлена после завершения работы функции. Т.е. изменение локали внутри потока повлияет на все остальные потоки.
Необходимо сделать:
Locale::Default::Get() сделать шаблонной (int) и при вызове в качестве параметра шаблона указывать id потока?
