Конструктор(ы)

SectionImpl::SectionImpl ( const String_t &  _Name)
explicit

Конструктор создания объекта раздела верхнего уровня.

  • Читает из xml-файла настроек (из раздела с указанным именем) значения параметров и рекурсивно создает иерархию вложенных разделов.
  • Если нет файла настроек или раздела с указанным именем, создает пустой раздел.
  • В разделе с указанным именем создает параметр 'ModuleLocation', который представляет собой полный путь к текущему модулю программы.
Аргументы
[in]_NameИмя дочернего узла узла settings файла настроек.

void SectionImpl::Save ( void  )
overridevirtual

Функция сохранения настроек в файл.

  • Выполняется только для корневого раздела настроек и рекурсивно сохраняет параметры дочерних разделов. Сохраняет настройки:
  • Которым были успешно установлены значения по умолчанию.
  • Измененные параметры.
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).

Замещает alicorn::settings::ISection.
