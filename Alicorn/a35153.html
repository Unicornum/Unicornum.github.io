Подробное описание

Класс входит в Settings
Класс доступа к настройкам раздела.

  • Предназначен для 'обычного' использования - доступа к настройкам раздела верхнего уровня и его дочерним разделам.
  • Объект раздела хранит объект корневого раздела, т.к. для работы класса реализации необходимо существование объекта корневого раздела все время, пока существует хотя бы один объект конкретного раздела настроек.

Конструктор создания раздела настроек верхнего уровня.

  • Под разделами верхнего уровня подразумеваются дочерние разделы корневого раздела.
  • Если в качестве корневого раздела задан nullptr, объект будет создан.
Аргументы
[in]_NameИмя раздела верхнего уровня (если такого раздела нет, будет создан пустой).
[in]_pRootSectionОбъект корневого раздела настроек (используется в подставном классе).

Функция для добавления особого параметра.

  • Назначение особых параметров - использование в качестве тегов для замены у других параметров.
  • Значения особых параметров можно установить только один раз и невозможно изменить (их можно только прочитать).
  • Особые параметры не будут сохранены в файле настроек.
Аргументы
[in]_NameИмя параметра.
[in]_ValueЗначение параметра.
Исключения
std::exception
  • Попытка добавить параметр, имя которого совпадает с именем уже существующего в текущем разделе параметра.

Установка параметру значения по умолчанию.

  • Значение устанавливается только для несуществующих параметров (т.е. эти параметры создаются).
