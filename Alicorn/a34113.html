Класс входит в Virtual File System
Класс реализации интерфейса преобразования для шифрования данных.

Параметры шаблона
TCryptoКласс, производящий преобразование данных (одна из специализаций класса alicorn::converter::Crypto).
TPasswordКласс для преобразования пароля (должен содержать статическую функцию Decrypt(), производящую преобразование пароля, обратное тому, что производит Encryptor внутри указанной в конструкторе dll).
