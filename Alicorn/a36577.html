Подробное описание

Класс входит в Version
Класс получения информации о версии модуля.

Windows:

  • Информация о версии извлекается из ресурсов указанного модуля (поэтому заголовочный файл можно включать куда угодно, изменение версии не приведет к перекомпиляции файлов, включающих этот заголовочный файл). Android:
  • Информация о версии возвращается как значение макросов из файла версии (изменение версии приведет к перекомпялиции всех файлов, в которые включен заголовочный файл).
