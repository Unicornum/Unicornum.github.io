Подробное описание

Класс входит в проект Localization
Класс получения строки для текущего выбранного языка программы.

  • Класс не предназначен для самостоятельного использования, обеспечивает логику работы сгенерированного утилитой Localization класса (подробнее см. Localization).

Конструктор класса.

  • Создает набор строк для выбранного в настройках языка (если для указанного языка нет файла локализации, будет использован файл Default.xml).
  • В настройках программы язык указывается как параметр Language раздела Localization.
