Подробное описание

Класс входит в Проект Document
Класс-обертка для работы с xml документом при помощи библиотеки RapidXml.

  • Источник: http://rapidxml.sourceforge.net/
  • Класс является оберткой для библиотеки rapidxml и не предназначен для самостоятельного использования, использовать следует класс Xml, указав ему в качестве параметра шаблона xml::RapidXml.
  • Описание функций см. у класса-интерфейса IXml.
  • Поддерживаются файлы/данные только в формате UTF8.
