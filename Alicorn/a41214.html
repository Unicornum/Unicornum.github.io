Подробное описание

Класс входит в проект Platform
Вспомогательный класс-обертка для вызовов функций java из кода С++.

  • Вызов функций класса безопасен в том смысле, что в случае, когда функция будет вызвана до инициализации внутреннего объекта java класса, она будет ждать инициализации и вернет управление, когда выполнение вызова java кода станет доступно (см. ниже особый случай с функцией JNI_OnLoad()).
    Заметки
    Функции класса нельзя использовать внутри функции JNI_OnLoad(), т.к. инициализация внутреннего объекта java класса произойдет после завершения ее работы; поэтому в случае необходимости инициализации логгера в функции JNI_OnLoad() (например, при запуске фонового сервиса), следует в ней запустить фоновый поток (который и произведет инициализацию, при вызове функций дождавшись инициализации внутреннего объекта) и завершить работу функции JNI_OnLoad().

    Пример использования

Для опеспечения работоспособности класса необходима его предварительная инициализация.

Проект Packaging

  • Добавить папку src/externals как свойство svn:externals с путем
    https://svn/svn/Prebuilt.no.backup/Solution/Properties/Utilities/Externals/Alicorn/_Root/Android
  • Подключить к проекту файлы из этой папки.
  • Добавить класс главной activity и загрузку главного модуля программы (предполагается, что главный модуль программы будет иметь имя файлa libExample.so):
    package com.example;
    public class MainActivity extends com.alicorn.MainActivity
    {
    static
    {
    System.loadLibrary("Example");
    }
    }
  • В файл AndroidManifest.xml добавить соответствующие записи:
    <activity android:name="com.example.MainActivity" ...>
    <meta-data android:name="android.app.lib_name" android:value="Example"/>
Заметки
При инициализации сервиса необходимо вручную создать объект класса com.alicorn.system.platform.Context, передав ему результат вызова функции getBaseContext():
public class BackgroundService extends Service
{
static
{
System.loadLibrary("Example");
}
@Override
public void onCreate()
{
new com.alicorn.system.platform.Context(getBaseContext());
...
