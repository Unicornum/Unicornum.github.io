Подробное описание

  • Файл предназначен для 'развязывания' проекта Test.Project (т.к. проект копируется в каждое решение) и конкретной реализации логики работы проекта.
  • Этот файл include'тся в файле stdafx.cpp проекта Test.Project, поэтому он должен содержать все файлы реализации, необходимые для работы этого проекта.
  • Содержимое файла может меняться (это затронет все тестовые проекты всех решений, которые его используют), но имя файла и его расположение меняться не должно.
