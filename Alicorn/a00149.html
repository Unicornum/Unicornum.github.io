Подробное описание

Класс входит в Проект Vfs
Класс реализации упаковки файлов в хранилище/получения содержимого файла из упакованного хранилища.

  • Хранилище может быть построено из файлов, расположенных в папке при помощи функции Build().

Стркуктура хранилища:

  • Заголовок.
  • Записи о файлах (каждая из которых представляет собой: простую структуру фиксированного размера, имя файла (в utf8->base64), содержимое файла).

void Pack::Build ( const Path_t &  _PathToSourceDirectory) const

Функция для построения хранилища из файлов в указанной папке.

  • Функция переберет все файлы в указанной папке (+ в подпапках рекурсивно) и построит хранилище, данные из которого можно будет извлекать функцией GetData().
  • Если в конструкторе задан набор преобразователей данных, он будет использован при упаковке (соответственно, для распаковки такого хранилища следует использовать точно такой же набор преобразователей).
  • Имена файлов внутри хранилища будут соответствовать путям исходных файлов внутри указанной папки.
  • Результат будет сохранен в файл, путь к которому задан в конструкторе.
  • Если папка, в которой должно быть создано хранилище, не существует, она будет создана.
Аргументы
[in]_PathToSourceDirectoryПуть к папке, содержимое которой будет упаковано.
Исключения
std::exception
  • Попытка упаковать несуществующую папку.
  • Файл, указанный в конструкторе, уже существует.
  • Не удалось открыть один из упаковываемых файлов.
  • Ошибка во время упаковки (подробнее см. описание исключения).

Функция получения содержимого указанного файла.

  • Функция предназначена извлечения данных конкретного файла из хранилища.
  • Хранилище рассматривается как папка, внутри которой расположены файлы (подробнее см. в описании функции Build()).
Аргументы
[in]_PathToFileПуть к искомому файлу внутри хранилища, указанного в конструкторе.
[out]_DataОбъект, в который будет записано содержимое файла, если таковой будет найден.
Возвращает
true Файл найден, данные прочитаны.
false
  • Указанного файла нет в текущем хранилище.
Исключения
std::exception
  • Структура хранилища не соответствует ожидаемой (подробнее см. описание исключения).
  • Файл хранилища, указанного в конструкторе, не существует или не может быть открыт для чтения.
