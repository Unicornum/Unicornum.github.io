Класс входит в проект Std.Memory
Класс буфера данных.

  • Класс предназначен для гарантированной передачи блока данных без копирования; передача данных между объектами класса возможна только перемещением.
  • Конструкто копирования (нужен для преобразования класса в std::any) также перемещает данные из старого в новый объект.
  • Для передачи данных стороннему коду поддерживается использование std::data() и std::size(), а также std::cbegin() и std::cend().
Параметры шаблона
TElementТип одного элемента данных, хранящихся в буфере.
