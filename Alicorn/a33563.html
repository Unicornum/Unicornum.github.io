◆ Text() [2/2]

alicorn::extension::std::Text::Text ( const Path_t &  _PathToFile,
bool  _IsEntirely = true 
)
explicit

Конструктор для загрузки текста из файла.

  • Загрузит текст из файла и разобьет его на отдельные строки.
  • Разделителем считается '\r\n'.
  • При преобразовании строк, прочитанных из файла будет использоваться текущая локаль по умолчанию (см. class Locale).
Аргументы
[in]_PathToFileПуть к файлу, который должен быть загружен.
[in]_IsEntirely
  • true - чтение файла целиком с последующим разбиением его на отдельные строки (более быстрый (в 20 и более раз) вариант, но требующия больше памяти).
  • true - на Android можно таким образом читать файлы из Assets.
  • false - чтение файла по одной строке (медленный вариант из-за преобразования кодировки, но требующий меньше памяти).
Исключения
std::exception
  • Не удалось открыть указанный файл.

◆ Save() [2/2]

void alicorn::extension::std::Text::Save ( const Path_t &  _PathToFile) const

Функция для сохранения текста в указанный файл.

  • Существующий файл будет перезаписан.
  • Если папки для указанного пути не существуют, они будут созданы.
  • Сохранение будет производиться по одной строке (медленно, но требует меньше памяти) или целиком (быстро, но займет больше памяти) в зависимости от флага, указанного в конструкторе.

[in]_PathToFileПуть к файлу, в который будет сохранен текст.
Исключения
std::exception
  • Не удалось открыть указанный файл.
