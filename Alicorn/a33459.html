Подробное описание

Класс входит в проект Document
Класс реализации для работы с простыми текстовыми документами.

  • Используется в качестве первого параметра шаблона класса alicorn::document::Universal.
  • Документ текстового файла представляет собой дерево, содержаще отдельные строки файла как дочерние узлы корневого узла (объекта документа), попытка использования дочерних узлов уровнем ниже первого приведет к исключению.
  • Доступ к отдельным строка файла производится через Value, а Name и атрибуты не имеют смысла (попытка чтения/записи имени или атрибута узла приведет к исключению).
  • Примеры использования см. Document.

Функция для сохранения текста в файл.

Функция для сохранения текста в указанный файл.

  • Сохранение будет производиться в тот же файл, из которого текст был загружен в конструкторе.
Исключения
std::exception
  • Объект класса был создан не из файла.
  • Не удалось открыть исходный файл.
  • Существующий файл будет перезаписан.
  • Если папки для указанного пути не существуют, они будут созданы.
  • Сохранение будет производиться по одной строке (медленно, но требует меньше памяти) или целиком (быстро, но займет больше памяти) в зависимости от флага, указанного в конструкторе.
Аргументы
[in]_PathToFileПуть к файлу, в который будет сохранен текст.
Исключения
std::exception
  • Не удалось открыть указанный файл.
