Класс входит в Проект Document
Интерфейсный класс для реализаций классов работы с xml документами.

  • Класс реализации, унаследованный от интерфейса, должен в конструкторе создавать валидный пустой документ.

pure virtual

Установить маркер текущего узла на первый дочерний элемент текущего узла.

  • Если текущий узел не содержит дочерних узлов, маркер должен остаться на текущем узле.

true Есть дочерние узлы.

false Текущий узел не содержит дочерних узлов.

pure virtual

Установить маркер текущего узла на следующий узел родительского узла текущего.
