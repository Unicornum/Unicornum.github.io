template<class TSource , class TContainer , class TPredicate >
Utilities::Text_t< TSource > alicorn::source::document::Utilities::Insert ( const Text_t< TSource > & _Text,
const TContainer & _Inserted,
TPredicate _Predicate )

Функция вставки набора строк в текстовый документ.

  • Результат будет возвращен в том же формате, что и исходный документ.
  • Исходный документ не будет изменен.
Аргументы
[in]_TextИсходный текстовый документ.
[in]_PredicateПредикат, который определяет, после какой строки будет вставлен указанный набор строк.
[in]_InsertedНабор вставляемых строк.
Возвращает
Текстовый документ, в котором после каждой строки исходного документа, для которой предикат вернет true, будет вставлен указанны набор строк.
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).

Функция замены во всем документе указанной строки на заданное значение.

  • Результат будет возвращен в том же формате, что и исходный документ.
  • Исходный документ не будет изменен.
Аргументы
[in]_TextИсходный текстовый документ.
[in]_PatternТекст, который следует заменить.
[in]_ChangeЗначение (любого типа, который поддерживает преобразование в String при помощи boost::lexical_cast<>()), на которое будет заменен искомый текст.
Возвращает
Текстовый документ, в котором произведены замены.
