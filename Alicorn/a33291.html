Класс входит в Std
Базовый класс для проверки кода ошибки.

  • Класс предназначен для построения макроса, бросающего исключение в случае, когда вызов функции завершился возвратом кода ошибки.
  • Класс-наследник должен определить статические функции IsSuccessfull() и GetMessageText(), получающими в качестве параметра переменную того же типа, который возвращает тестируемая функция (см. пример использования).
Параметры шаблона
Вкачестве параметра шаблона базовому классу нужно указать сам класс.

Создание макроса для конкретного случая:

// Создаем класс-наследник, определяем нужные функции:
class TestCallProxy : public alicorn::extension::std::CallProxy<TestCallProxy>
{
public:
// Функция должна вернуть false, если _ErrorCode содержит код ошибки.
inline static bool IsSuccessfull(const int _ErrorCode)
{
return (_ErrorCode > 0);
}
// Функция должна вернуть текстовое описание ошибки, соответствующее _ErrorCode
inline static std::string GetMessageText(const int _ErrorCode)
{
static const char * Messages[] =
{
"Fail: 0",
"Fail: -1",
"Fail: -2"
};
return Messages[abs(_ErrorCode)];
}
};
// Объявляем макрос, который будет использоваться для проверки кода ошибки,
// возвращаемого функцией:
#define TEST_CALL \
CALL_PROXY(TestCallProxy)

Использование созданного макроса:

// Функция должна быть объявлена как int CallTestedFunction(void);
TEST_CALL CallTestedFunction();
