Класс входит в Patterns
Фабрика объектов.

  • Предназначена для создания объектов по идентификатору или строковому имени класса
  • Доработанный вариант, взятый отсюда.
Параметры шаблона
TInterfaceКласс интерфейса, в указателях на который фабрика будет создавать объекты.
TIdКласс идентификатора класса, при помощи которых будут создаваться объекты (требутся оператор 'ostream <<' для добавления идентификатора в сообщение об ошибке).
TArgsАргументы, передаваемые конструктору класса создаваемого объекта.
