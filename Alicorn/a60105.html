Virtual File System

Список классов проекта: Virtual File System

Введение

Проект виртуальной файловой системы, реализующей однотипное получение данных, сходное с чтением данных из файлов.

Подробное описание проекта

Виртуальная файловая система позволяет абстрагироваться от способа хранения данных, предоставляя интерфейс получения этих данных, а также добавлять необходимые реализации доступа к данным.
Также, виртуальная файловая система позволяет получать данные из различных источников (например, ее можно настроить таким образом, чтобы в отладочной версии программа получала данные с диска, а в релизе - из БД).

Доступные реализации

  • Чтение файлов с диска.
  • Получение данных из файлов, упакованных и зашифрованных в специальное хранилище.
