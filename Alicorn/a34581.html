◆ USING_MOCK

#define USING_MOCK

Макрос входит в Testing
Макрос для объявления типов, которые в тестовом проекте должны использоваться из пространства имен mock, а в основном - из глобального пространства имен.

Пример использования:

// В основном проекте Type_t будет ::boost::Type;
// В тестовом проекте Type_t будет ::mock::boost::Type;
using Type_t = USING_MOCK ::boost::Type;
