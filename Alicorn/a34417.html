template<class TImplementation, class TFileData>
class alicorn::document::Universal< TImplementation, TFileData >

Класс входит в проект Document
Класс-обертка для унифицированной работы с различными типами документов.

  • Доступные виды документов:
    • alicorn::document::Text.
    • alicorn::document::RapidXml.
  • Документ рассматривается как иерархическое дерево, состоящее из узлов, каждый из которых также содержит дочерние узлы.
  • Объект документа явлюяется корневым узлом документа.
Параметры шаблона
TImplementationКласс реализации для работы с конкретным типом документа (один из классов Implementation).
TFileDataКласс для доступа к данным (один из классов Data).
