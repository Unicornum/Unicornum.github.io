Подробное описание

Класс входит в Std
Класс универсальных Unicode строк.

  • Предназначен для использования подхода "используем внутри проекта строки в удобной нам кодировке, преобразуя их в нужные при взаимодействии со сторонним кодом" (для преобразований строк см. функцию string_cast<>()).
  • Допускает предварительное объявление (использовать файл <alicorn\extension\std\string.forward.hpp>).
  • Для реализации функционала используются строки из стандартной библиотеки шаблонов С++, а также алгоритмы работы со строками из Boost.
  • Содержит внутри себя строку в формате Unicode в зависимости от того, каким образом эта строка была создана (см. макросы u8T(), uT(), u16T(), u32T()).
  • Удобный интерфейс, запрещающий использовать класс строки как контейнер.
  • Поддержка шаблона std::hash<>().
  • Поддержка преобразования строк в другие типы при помощи boost::lexical_cast<>().
  • Поддержка сериализации при помощи boost::serialization.
  • Примеры использования см. в описании группы String.
