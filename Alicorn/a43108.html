Класс входит в Std
Класс исключения, предназначенный для сигнализирования о том, что работа программы должна быть завершена.

  • Класс намеренно не наследуется от std::exception, чтобы исключение не перехватывалось 'попутно'.
  • Используется для завершения программы из любого места программы (например, это исключение бросает логгер при записи сообщения, помеченного как Fatal.
  • Предполагается, что программа в главном цикле будет перехватывать это исключение и завершать работу корректным образом.

Пример обработки исключения при использовании wxWidgets

namespace app
{
class Main final :
public wxApp
{
// ... другой код
public:
bool OnExceptionInMainLoop(void) override
{
try
{
// Повторяем произошедшее исключение.
throw;
}
catch (const ::alicorn::extension::std::exception::Abort &)
{
// Завершаем работу программы
wxExit();
}
catch(...)
{
return wxApp::OnExceptionInMainLoop();
}
return true;
}
// ... другой код
};
} // namespace app
