Класс входит в проект Document
Интерфейс классов, предоставляющих доступ к содержимому файла.

  • Классы реализаций должны иметь два конструктора:
    • Конструктор по умолчанию, создающий пустой набор данных.
    • Конструктор, получающий в качестве параметра путь к файлу, читающий содержимое этого файла.
