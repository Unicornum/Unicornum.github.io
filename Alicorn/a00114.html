Info::Info ( void  )

Конструктор получения информации для текущего модуля программы.

  • Если создать объект внутри .exe - информация будет извлечена из этого файла, внутри .dll - из нее.
Исключения
std::exception
  • Текущий модуль не содержит ресурса версии.
