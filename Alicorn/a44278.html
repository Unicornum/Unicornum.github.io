pure virtual

Функция получения содержимого 'файла'.

Аргументы
[in]_PathToFileПуть к нужному файлу. Это может быть путь к файлу на диске или идентификатор искомого ресурса (подробнее см. у конкретных реализаций).
[out]_DataОбъект, который должен быть заполнен данными из файла, если таковой существует.
Возвращает
true Указанный файл найден, успешно прочитан и _Data содержит данные из этого файла.
false Указанный файл не найден, _Data содержит невалидные данные.

Замещается в alicorn::modules::vfs::Pack и alicorn::modules::vfs::FileSystem.
