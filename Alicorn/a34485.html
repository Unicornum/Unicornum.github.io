Класс входит в проект Document
Интерфейсный класс классов реализаций работы с документами.

  • Предоставляет интерфейс для классов, которые могут использоваться 'под капотом' класса alicorn::document::Universal.
  • Класс реализации должен иметь конструктор, принимающий в качестве параметра объект std::shared_ptr<alicorn::document::data::IData>, который предоставляет доступ к содержимому исходного файла.
