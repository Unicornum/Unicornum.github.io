Класс входит в Patterns
Фабрика объектов (создание объектов по идентификатору (строке) класса).

Доработанный вариант, взятый отсюда .

Параметры шаблона
TInterfaceКласс интерфейса, в указателях на который фабрика будет создавать объекты.
TIdКласс идентификатора класса, при помощи которых будут создаваться объекты (требутся оператор 'ostream <<' для добавления идентификатора в сообщение об ошибке).

Пример использования:

Регистрация классов в фабрике:

using String_t = ::alicorn::extension::std::String;
class Type
{
public:
enum Value
{
Der1 = 0,
Der2
};
};
class IBase :
public ::alicorn::patterns::factory::Registrator<IBase, Type::Value>
{
public:
virtual String_t Test(void) const = 0;
virtual ~IBase() {}
};
class Der1 final :
public IBase
{
public:
String_t Test(void) const override { return uT("Der1::Test()"); }
};
FACTORY_REGISTER(Der1, Type::Der1);
class Der2 final :
public IBase
{
public:
String_t Test(void) const override { return uT("Der2::Test()"); }
};
FACTORY_REGISTER(Der2, Type::Der2);

Создание объектов зарегистрированных классов

auto pObject = factory::make_unique<IExample_t>(::example::Type::Der1);
EXPECT_EQ(uT("Der1::Test()"), pObject->Test());
pObject = factory::make_unique<IExample_t>(::example::Type::Der2);
EXPECT_EQ(uT("Der2::Test()"), pObject->Test());
