Класс входит в Virtual File System
Класс реализации получения содержимого файлов с диска.

Android:

  • Для доступа к файлам из assets следует в качестве пути к папке передать в конструктор CurrentModule::GetAppRootPath().

Функция получения содержимого указанного файла.

Аргументы
[in]_PathToFileПуть к искомому файлу (внутри папки, заданной в конструкторе).
[out]_DataОбъект, в который будет записано содержимое файла, если таковой будет найден.
Возвращает
true Файл найден, данные прочитаны.
false
  • Файл не существует.
  • Файл существует, но не может быть открыт для чтения.
