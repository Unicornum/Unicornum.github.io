Проект логики работы утилиты, генерирующей заголовочный файл, содержащий бинарные данные указанного файла.

Проект Localization.App

Проект логики работы утилиты, генерирующей заголовочный файл с идентификаторами строк, используемыми для локализации программы.

Проект Version.App

Проект логики работы утилиты, формирующей файл со строками версии решения для включения его в проект Packaging для Android.

Проект логики работы утилиты для обработки результатов вычисления метрик кода в решении.
