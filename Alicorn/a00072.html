Подробное описание

Класс входит в Testing.App
Класс построения отчета о прогоне тестов для документации.

  • Отчет добавляется в файл, путь к которому указан в конструкторе.
  • Если файл не существует, будет создана заготовка страницы отчета с именем SolutionRunTestsPage.
  • Новый отчет будет добавлен в начало страницы.
  • Цвет фона записи будет зависеть от того, есть проваленные тесты или нет.
