Подключение библиотек к решению

  • Скачать нужную версию Alicorn SDK.
  • Распаковать архив в папку (например) E:\SDK\Alicorn\vXXX и выполнить в этой папке cmd команду
    setx UNICORNUM_ALICORN_SDK_VXXX %CD%
  • В корень решения добавить файл Directory.Build.targets (рядом с файлом .sln), содержащий ..\Example.Directory.Build.targets это автоматически даст доступ всем проектам этого решения к собранным библиотекам именно этой версии.
Заметки
Вторая строка Import позволяет использовать props файл, выводящий сообщение об ошибке, если ожидаемая версия Alicorn SDK не найдена.
