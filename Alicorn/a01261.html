void boost::filesystem::copy_all ( const path &  _From,
const path &  _To 
)

Функция для копирования содержимого папки, включая файлы и дочерние папки.

  • Функция рекурсивно копирует содержимое исходной папки в папку результата.
Аргументы
[in]_FromИсходная папка.
[in]_ToПапка результата.
Исключения
std::exception
  • Путь к исходной папке не существует.
  • Исходный путь не является путем к папке.

Функция вычисления относительного пути для двух указанных полных путей к папкам.

Аргументы
[in]_FromПолный путь к исходной папке.
[in]_ToПолный путь к папке, к которой будет вычисляться относительный путь.
std::vector< uint8_t > boost::filesystem::load_binary_file_from_disk ( const path &  _PathToFile)

Функция получения содержимого файла с диска по указанному пути.

Исключения
std::exception
  • Указанный файл недоступен для чтения.
  • Размер файла слишком велик (т.е. для 32-хбитных систем > 4 Гб).
