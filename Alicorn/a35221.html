Подробное описание

Класс входит в Settings
Класс доступа к настройкам раздела.

  • Предназначен для 'особого' использования - полного доступа к настройкам раздела верхнего уровня и его дочерним разделам.
  • Объект раздела хранит объект корневого раздела, т.к. для работы класса реализации необходимо существование объекта корневого раздела все время, пока существует хотя бы один объект конкретного раздела настроек.
