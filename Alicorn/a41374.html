Подробное описание

Класс входит в проект ToolsProcessingMetricsAppPage
Класс обработки суммарного результата вычисления метрик кода в решении.

  • Извлекает данные из файла Metrics.common.xml в указанной в конструкторе папке и добавляет запись (в начало файла) в файл Metrics.result.html.
  • Осуществляет ротацию записей в файле Metrics.result.html, оставляя 20 последних.
  • Если файл Metrics.result.html не существует, он будет создан.
