Проект Logger

Библиотека логирования программы предназначена для ведения журнала работы программы.
Запись сообщений в журнал производится путем добавления в код программы записей вида:

// Записать в лог сообщение "Найдено 5 элементов"
LOGGER(Warning) << uT("Найдено ") << 5 << " элементов";

Страница проекта: Logger

Проект Vfs

Страница проекта: Virtual File System
