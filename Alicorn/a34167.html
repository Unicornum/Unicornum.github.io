◆ ALICORN_UNUSED

#define ALICORN_UNUSED (   ParameterName)

Макрос входит в Cpp
Макрос для обозначения неиспользуемых параметров функций.

  • При компиляции раскрывается в пустую строку, чтобы избежать предупреждений о неиспользуемом параметре.
  • При генерации документации раскрывается в имя параметра (см. файл настроек для Doxygen), что позволяет использовать имя параметра в описании функции.

◆ PROPERTY_GET

#define PROPERTY_GET (   ReturnedType,
  Name 
)    ReturnedType Get##Name(void) const

Макрос входит в Cpp
Макрос объявления свойства, доступного для чтения.

  • Объявление функции внутри макроса сделано, чтобы избавиться от предупреждений CppCheck о неиспользуемой функции.
  • Описание функции в .cpp файле, объявленной таким образом не попадет в документацию, поэтому документировать ее следует в объявлении класса.

Пример использования:

// Объявление свойств
class PropertyGet
{
public:
// Реализация 'на месте'
PROPERTY_GET(int, Int) const {return m_Int;}
// Только объявление
PROPERTY_GET(long, Long) const;
private:
int m_Int;
long m_Long;
};
// Реализация ранее объявленного свойства
long PropertyGet::GetLong(void) const
{
return m_Long;
}
// Использование:
PropertyGet Example;
int Int = Example.Int;
long Long = Example.Long;

◆ PROPERTY_SET

#define PROPERTY_SET (   SetType,
  Name 
)    void Set##Name(const SetType & _Value)

Макрос входит в Cpp
Макрос объявления свойства, доступного для записи.

  • Обявление функции внутри макроса сделано, чтобы избавиться от предупреждений CppCheck о неиспользуемой функции.
  • Имя устанавливаемого значения в inline реализации - _Value.
  • Описание функции в .cpp файле, объявленной таким образом не попадет в документацию, поэтому документировать ее следует в объявлении класса.

Пример использования:

// Объявление свойств
class PropertySet
{
public:
// Реализация 'на месте'
PROPERTY_SET(int, Int) {m_Int = _Value;}
// Только объявление
PROPERTY_SET(long, Long);
private:
int m_Int;
long m_Long;
};
// Реализация ранее объявленного свойства
void PropertySet::SetLong(const long & _Value)
{
m_Long = _Value;
}
// Использование:
PropertySet Example;
Example.Int = 0;
Example.Long = 0;
