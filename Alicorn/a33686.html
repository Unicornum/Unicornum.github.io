◆ operator[]() [3/3]

template<class TIndex>
Node< TIndex > alicorn::document::Node< TIndex >::operator[] ( const String_t &  _Id)

Оператор получения объекта дочернего узла текущего.

  • Если у текущего узла есть несколько дочерних узлов с указанным идентификаторм, будет возвращен первый найденный.
  • Если дочернего элемента с указанным идентификатором нет, он будет создан.
Аргументы
[in]_IdИдентификатор узла (для xml - имя узла, для текстового документа не имеет смысла).
Возвращает
Node Объект найденного/созданного узла.
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).
