Исходные требования

Требования первого этапа

  • Диалоговое окно отображения результатов прогона тестов.
  • Цвет фона всех вкладок окна должен зависеть от результата прогона тестов (зеленый, если все тесты пройдены успешно, красный, если есть хотя бы один проваленный тест).

Диалоговое окно отображения результатов прогона тестов

  • Вкладка общей статистики.
  • Вкладка со списком всех тестов.
  • Вкладка со списком проваленных тестов.

### Вкладка общей статистики

  • Общее количество выполненных тестов.
  • Количество успешно пройденных тестов.
  • Количество проваленных тестов.
  • Количество отключенных тестов.
  • Дата/время запуска тестов.
  • Название и версия используемой библиотеки тестирования.

### Вкладка со списком всех тестов

  • В виде иерархического дерева: модули/классы/тесты.
  • Узлы дерева должны помечаться иконками успеха/провала.

### Вкладка со списком проваленных тестов

  • В виде иерархического дерева: модули/классы/тесты.
  • Отображение подробного описания при выборе проваленного теста.
