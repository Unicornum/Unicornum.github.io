Класс входит в проект Std
Простой класс-обертка для указателя.

  • Простая обертка над указателем, который инициализируется в конструкторе и гарантированно уничтожается в деструкторе.
  • Понадобился из-за того, что при использовании std::unique_ptr<> нужно указывать в заголовочном файле "#include <memory>"; должен использоваться исключительно для реализации паттерна pImpl.
Параметры шаблона
TКласс объекта, который будет содержаться внутри Ptr.

Example.hpp

#pragma once
#include <alicorn\patterns\pimpl.hpp>
class Example final
{
class Impl;
public:
int GetIndex(void) const;
private:
::alicorn::patterns::pimpl::Ptr<Impl> m_Impl;
public:
Example(void);
~Example(void);
};

#include "stdafx.h"

public:

}

int Example::GetIndex(void) const
{
return m_Impl.Get().m_Index;
}
