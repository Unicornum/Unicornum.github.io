Функция входит в Std
Функция преобразования строк из одной кодировки в другую.

  • Функция представляет собой шаблон, предоставляющий интерфейс для преобразования двубайтовых и однобайтовых строк (реализация преобразования расположена в классе StringCastProxy<>).
  • Функция предназначена для поддержки работы класса alicorn::extension::std::String и позволяет производить следующие виды преобразований (для несуществующих будет брошено исключение):
  • String <-> std::u32string.
  • String <-> std::u16string.
  • String <-> std::wstring.
  • String <-> std::string(1251).
  • String <-> std::string(866).
  • String <-> std::string(utf8).
  • String <-> boost::filesystem::path (требуется включение файла <alicorn\extension\boost\string-cast.hpp>).
  • String <-> CString (требуется включение файла <alicorn\extension\mfc\string-cast.hpp>).
  • String <-> wxString (требуется включение файла <alicorn\extension\wx.hpp>).
Параметры шаблона
TTargetКласс строки, которую нужно получить.
TLocaleКласс используемой кодировки (при преобразовании std::string в String это кодировка исходной строки, при обратном - кодировка результата).
Для задания кодировки использовать дочерние классы класса Locale.

Пример использования:

  • Вспомогательный шаблон для преобразования String в/из различных форматов UTF (поддерживаются UTF16, UTF32).
