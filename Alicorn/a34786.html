class alicorn::extension::std::Locale::Scope::SetDefault< TLocale >

Класс установки локали по умолчанию для текущей области видимости.

  • Установленная таким образом локаль будет активной до следующей закрывающей фигурной скобки, после чего будет восстановлена предыдущая локаль.
Аргументы
[in]_NameИмя локали.
[in]_CategoryКатегория локали.
Исключения
std::exception
  • Указанная локаль недоступна.
