Сохранение документа в xml файл.

  • При вызове функции без параметров, документ будет сохранен в тот же файл, из которого он был загружен изначально.
Аргументы
[in]_PathToFileПуть к файлу, в который следует сохранить документ.
Исключения
std::exception
  • Указанный файл открыть для записи не удалось.
  • Документ был создан не из файла.
