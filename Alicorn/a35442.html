Testing.App

Список классов проекта: Testing.App

Введение

Проект, содержащий реализацию проектов тестирования.

Подробное описание проекта

Исходные требования

Проект реализует программу отображения результат прогона тестов.
Программа существует в трех версиях:

  • Тесты одного класса.
  • Тесты всех классов одного проекта.
  • Тесты всех классов всех проектов решения.

Тестовый проект одного класса

Собирается как исполняемый файл, при запуске отображает результат прогона тестов своего класса.

Схема взаимодействия классов тестового проекта одного класса

Общий тестовый проект проекта

Собирается как dll, предоставляя интерфейс получения результата прогона тестов всех классов одного проекта.

Схема взаимодействия классов тестового проекта всех классов проекта

Тестовый проект всего решения

Собирается как исполняемый файл, при запуске собирает и отображает информацию о прогоне тестов всех проектов решения.

Схема взаимодействия классов тестового проекта всего решения
