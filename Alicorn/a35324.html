Класс входит в проект Std
Класс доступа к общему для всей программы объекту.

  • Предназначен для доступа из любой точки программы к единственному общему объекту программы.
  • В класс, который будет использоваться как синглетон следует добавить функцию создания общего используемого объекта (эта функция вызывается один раз при первом обращении к объекту).
Параметры шаблона
TUsingКласс общего используемого объекта с модификаторами использования (именно в указанном виде созданный объект будет возвращаться функцией GetInstance()).

Пример использования

Объявление класса

class Example
{
public:
int GetValue(void) const { return m_Value; }
private:
const int m_Value;
private:
explicit Example(int _Value) : m_Value(_Value) {} // Закрытый!
friend ::alicorn::extension::std::Singleton<Example>;
};

Специализация функции создания объекта

namespace alicorn
{
namespace extension
{
namespace std
{
::std::unique_ptr<Example> Singleton<Example>::Make(void)
{
return ::std::unique_ptr<Example>{ new Example{ 1803221845 } };
}
} // namespace std
} // namespace extension
} // namespace alicorn

Использование единственного объекта

using Example_t = ::alicorn::extension::std::Singleton<const Example &>;
const auto Result1 = Example_t::GetInstance().GetValue();
using ExamplePtr_t = ::alicorn::extension::std::Singleton<Example *>;
const auto Result2 = ExamplePtr_t::GetInstance()->GetValue();

inlineprivate

Функция создания объекта синглетона.

  • Вызывается один раз при первом обращении к объекту синглетона.
  • Реализация по умолчанию создает объект при помощи конструктора без параметров, при необходимости задействовать другой конструктор вызывающий код может определить специализацию этой функции для нужного класса.
