Класс входит в Converter
Класс, определяющий общий интерфейс для кодирования/декодирования данных.

Класс не предназначен для непосредсвенного использования, используются конкретные специализации шаблона класса.

Параметры шаблона
TImplementationОдин из следующих классов для специализации шаблона:
  • AES256.
  • Base64.
  • PlantUml64.
