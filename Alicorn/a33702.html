class alicorn::document::Value< TIndex >

Класс входит в проект Document
Вспомогательный класс для доступа к значению узла.

  • Для текстового файла значением узла будет отдельная строка текста.
  • Для xml документа значением узла будет текст внутри тегов текущего узла (<entity>value</entity>).
