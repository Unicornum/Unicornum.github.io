Макрос входит в Std
Макрос для "бросания" исключения, содержащего расширенное описание ошибки.

  • Описание исключения будет содержать имя файла и номер строки места, где это исключение было выброшено.
  • При помощи опертора << в сообщение исключения может быть добавлена любая информация.
  • Тип бросаемого исключения передается в качестве параметра макроса.
  • Класс бросаемого исключения должен в конструкторе принимать строку как 'const char *'.

Пример использования:

try
{
// Следующая запись - 123-я строка в файле FileName.cpp:
throw STD_EXCEPTION << "Index: " << 5;
}
catch(const std::exception & _Ex)
{
// _Ex.what() вернет строку "FileName.cpp (123): Index: 5"
}
