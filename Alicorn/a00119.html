Подробное описание

Класс-обертка для классов конкретных кодировок.

Класс предназначен для:

  • Установки локали по умолчанию (глобальной или для текущей области видимости).
  • Указания используемой кодовой страницы при преобразовании строк при помощи функции string_cast<>().
Заметки
Установка локали для текущей области видимости не означает, что локаль устанвливается только для этой функции! Это знчит, что устанавливается глобальная локаль, которая будет восстановлена после завершения работы функции. Т.е. изменение локали внутри потока повляет на все остальные потоки.
Необходимо сделать:
Locale::Default::Get() сделать шаблонной (int) и при вызове в качестве параметра шаблона указывать id потока?

Примеры использования:

using namespace ::alicorn::extension::std;
// Установка глобальной локали по умолчанию:
// Теперь во всем коде текущего модуля будет использоваться
// кодовая страница ru_RU.CP1251
Locale::Global::SetDefault<Locale::Russian::CP1251>();
// Установка локали по умолчанию для текущей области видимости:
// Кодовая страница ru_RU.CP1251 будет использоваться до закрывающей
// фигурной скобки
Locale::Scope::SetDefault<Locale::Russian::CP1251> Dummy;
// Использование локали по умолчанию для работы с потоками.
::std::wstringstream Stream;
Stream << Locale::Default() << 123.456; // В поток запишется "123,456"
