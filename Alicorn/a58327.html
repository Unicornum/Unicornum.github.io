Функция загрузки значения одного параметра из настроек программы.

  • Если указанного параметра нет, останется значение по умолчанию.
  • Если значение было успешно прочитано из настроек, его тип будет помечен таким образом, чтобы его было невозможно изменить в дальнейшем.
Аргументы
[in]_SectionРаздел настроек программы, из которого будут читаться параметры.
[in]_NameИмя параметра.
