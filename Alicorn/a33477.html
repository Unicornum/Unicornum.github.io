◆ unique_ptr

template<class T >
using alicorn::extension::std::unique_ptr = typedef ::std::unique_ptr<T, void(*)(T *)>

Класс входит в проект Std
Вспомогательное определение аналога std::unique_ptr<> для захвата ресурсов.

  • В качестве deleter'a можно задавать лямбда-выражение (std::function<void(T *)> не используется, т.к. в этом случае вызывается его оператор присваивания при создании объекта unique_ptr<>, что существенно сказывается на времени создания объекта из пула).
  • Если при создании объекта будет возвращен nullptr, deleter вызван не будет.
Параметры шаблона
TКласс, в указателе на который будут храниться созданные объекты.

Вспомогательная функция создания объектов std::shared_ptr<> из.

  • Позволяет создавать корректные объекты 'обычных' std::shared_ptr<> из специализированной версии unique_ptr<>.

◆ make_unique() [1/2]

template<class T , class ... TArgs>
unique_ptr<T> alicorn::extension::std::make_unique ( TArgs &...  _Args)
inline

Функция входит в проект Std
Вспомогательная функция создания объектов, аналогичных std::unique_ptr<>.

  • Фактически, создана исключительно для того, чтобы быть задействованной в фабрике объектов (т.к. та требует две функции (создание объектов в куче и пуле), которые должны возвращать однотипные объекты).
Параметры шаблона
TКласс, объект которого должен быть создан.
TArgsАргументы, которые будут передаваться конструктору создаваемого объекта.
