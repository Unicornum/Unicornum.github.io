Класс входит в проект Std
Простой класс-обертка для указателя.

  • Простая обертка над указателем, который инициализируется в конструкторе и гарантированно уничтожается в деструкторе.
  • Понадобился из-за того, что при использовании std::unique_ptr<> нужно указывать в заголовочном файле "#include <memory>";
  • Должен использоваться только в особых случаях (создавался для реализации паттерна pimpl).
Параметры шаблона
TКласс объекта, который будет содержаться внутри объекта класса.
Заметки
Если требуется поддержка копирования, должен содержать функцию Clone(), возвращающую копию объекта (создание копий объектов должно производиться тем же способом, что и создание исходных объектов, т.е. если исходный объект был создан из пула памяти, то и копия должна создаваться в том же пуле)
class RealClass
{
public:
trivial_ptr<RealClass> Clone(void) const
{
return make_trivial<RealClass>(*this);
}
