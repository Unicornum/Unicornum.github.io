Макрос входит в Проект Testing
Макрос блокирования указанного файла.

  • Путь к файлу - объект класса boost::filesystem::path.
  • Макрос блокирует указанный файл таким образом, что его невозможно открыть.
  • Файл блокируется до ближайшей закрывающей фигурной скобки.

Функция побайтового сравнения двух файлов.

  • Вспомогательная функция для работы макросов сравнения двух файлов.
  • Функция предназначена для сокрытия ссылки на boost::filesystem.
Аргументы
[in]_PathToExpectedFileПуть к ожидаемому файлу.
[in]_PathToResultFileПуть к проверяемому файлу.
Возвращает
true Файлы совпадают.
false Файлы отличаются.
Исключения
std::exception
  • Один или оба указанные файлы открыть не удалось.
