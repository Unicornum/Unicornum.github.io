◆ BrowsingDirectories()

void Core::BrowsingDirectories ( const Path_t & _Path,
const IImplementation::Callback_t & _Callback ) const

Функция перебора 'файлов' в указанной папке.

  • Нужно учитывать, что перебор файлов будет производиться для всех добавленных в конструкторе реализаций; если реализации дублируют структуру файлов друг друга, вызов функтора обратного вызова будет также продублирован.
Аргументы
[in]_PathПуть к интересующей папке.
[in]_CallbackФунктор обратного вызова, который будет вызван для каждой папки, существующей в указанной папке.

◆ GetData()

Core::Data_t Core::GetData ( const Path_t & _PathToFile) const

Функция получения данных указанного файла.

  • Поиск данных указанного файла будет производиться в том порядке, в котором реализации были добавлены в конструкторе.
Аргументы
[in]_PathToFileПуть к искомому файлу (интерпретация этого параметра у различных реализаций может быть различной, см. подробнее у конкретного класса реализации).
Возвращает
Data Бинарные данные указанного файла (один в один, т.е. размер буфера будет равен размеру файла).
Исключения
std::exception
  • Указанный файл не был найден ни одной из заданных в конструкторе реализаций.
