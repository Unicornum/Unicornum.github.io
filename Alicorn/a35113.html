Введение

Раздел проектов вспомогательных утилит.

Подробное описание

Проекты этого раздела экспортируются в Externals и могут быть использованы в других решениях 'как есть'.

Проекты раздела

Проект Settings

Проект программы редактирования настроек.

  • При экспорте репозитория External окажется в решении и должен подключаться к нему, чтобы содержать правильное содержимое в ресурсе версии.
  • При запуске программа открывает настройки решения, в рамках которого была собрана и позволяет редактировать значения параметров.

Проект Version

Проект утилиты формирования файла с версией решения, предназначенный для включения в проект Package Android программы.

  • При экспорте репозитория External окажется в решении и должен подключаться к нему, чтобы содержать правильное содержимое в ресурсе версии.
  • Один из проектов, от которого зависит Packaging, должен в качестве события после построения запускать утилиту, передавая ей в качестве первого параметра командной строки путь к файлу, в котором будут формироваться строки версии решения, второго - путь к файлу AndroidManifest.xml проекта.
  • Чтобы строки версий из сформированного таким образом файла можно было использовать, он должен формироваться в папке res/values проекта Packaging.
  • Строки версий решения можно использовать следующим образом - в файле AndroidManifest.xml изменить:
    <manifest ...
    android:versionName="@string/app_version"
    ...
  • В файл AndroidManifest.xml будет вписываться текущая ревизия решения:
    <manifest ...
    android:versionCode="CURRENT_REVISION">
    ...
  • Образец формируемого файла со строками версии:
    <?xml version="1.0" encoding="utf-8"?>
    <resources>
    <string name="app_version">1.2.3.2015</string>
    <string name="app_name">Название программы (debug)</string>
    <integer name="app_revision">456</integer>
    </resources>
