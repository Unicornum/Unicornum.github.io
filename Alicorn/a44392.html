Класс входит в проект Details
Базовый класс реализации доступа к данным.

  • Пустой путь к файлу создает пустой набор данных.
  • Класс-наследник должен реализовать функции DoReadData() и DoSaveData() для реализации доступа к данным, хранящимся определенным образом.
Параметры шаблона
TSignatureКласс сигнатуры, которая будет проверяться при загрузке данных:
  • Default.
  • Utf8.
