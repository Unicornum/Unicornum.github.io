SectionImpl::SectionImpl ( void  )

Конструктор создания объекта корневого раздела.

  • Читает из xml-файла настроек значения параметров и рекурсивно создает иерархию вложенных разделов.
  • Если нет файла настроек или раздела с указанным именем, создает пустой раздел.

Создать особый параметр.

  • Назначение особых параметров - использование в качестве тегов для замены у других параметров.
  • Значения особых параметров можно установить только один раз и невозможно изменить (их можно только прочитать).
  • Особые параметры не будут сохранены в файле настроек.
Аргументы
[in]_NameИмя параметра.
[in]_ValueЗначение параметра.
Исключения
std::exception
  • Попытка добавить параметр, имя которого совпадает с именем уже существующего параметра.

Функция сохранения настроек в файл.

  • Выполняется только для корневого раздела настроек и рекурсивно сохраняет параметры дочерних разделов. Сохраняет настройки:
  • Которым были успешно установлены значения по умолчанию.
  • Измененные параметры.
Исключения
