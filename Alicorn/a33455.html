Класс входит в проект Document
Класс реализации работы с документом xml.

  • В качестве реализации используется библиотека [RapidXml].
  • Используется в качестве первого параметра шаблона класса alicorn::document::Universal.
  • Поддерживается работа только с файлами в формате utf8.
