Подробное описание

Класс входит в wxWidgets
Класс для создания изображений из набора байт.

  • Исходный набор байт должен содержать простую последовательность троек байт для каждого пикселя (порядок следования байт в тройке - BGR); именно в такой последовательности генерирует набор байт из 24-битного bmp файла утилита ConverterFileToRes или команда надстройки.

◆ wxImageFromRawData()

alicorn::extension::wx::wxImageFromRawData::wxImageFromRawData ( const Data_t & _Data,
const wxSize & _Size,
RevertLines  )

Конструктор класса.

  • В качестве исходного буфера будет использован буфер от последнего байта до нужного для указанных размеров изображения (т.е. заголовок bmp файла будет проигнорирован).
  • При загрузке изменит порядок следования строк в исходном изображении на обратный.
Аргументы
[in]_DataИсходный набор байт, содержащий 24-хбитную картинку.
[in]_SizeРазмеры изображения в пикселях.
Исключения
std::exception
  • Размер исходного набора байт меньше, чем требуется для изображения указанного размера.
