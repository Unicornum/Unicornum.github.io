const String_t & _Extension )
protected

Функция для формирования имени файла журнала, в который будут добавляться записи.

  • Функция создаст папку, в которой будут создаваться журналы, если ее нет.
  • Функция добавит к имени файла следующий номер, если файл журнала с таким именем уже существует.
Аргументы
[in]_FileNameИмя файла создаваемого журнала.
[in]_ExtensionРасширение (с точкой) файла создаваемого журнала.
Возвращает
Сформированное имя файла журнала с полным путем к нему.
Необходимо сделать
"Класс File не имеет права обращаться к конфигуратору напрямую!" ;
