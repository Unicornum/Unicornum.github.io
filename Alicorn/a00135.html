alicorn::extension::std::Text::Text ( const Path_t &  _PathToFile)
explicit

Конструктор для загрузки текста из файла.

  • Загрузит текст из файла и разобьет его на отдельные строки.
  • Разделителем считается '\r\n'.
  • При преобразовании строк, прочитанных из файла будет использоваться локаль, установленная через setlocale() (по умолчанию это системная локаль).
Аргументы
[in]_PathToFileПуть к файлу, который должен быть загружен.
Исключения
std::exception
  • Не удалось открыть указанный файл.
Необходимо сделать:
Локаль создается для каждой строки - это очень медленно, лучше прочитать файл целиком и добавить его через функцию Add()

void alicorn::extension::std::Text::Insert ( const String &  _Source,
Size_t  _Index 
)

Функция вставки текста на место указанной строки.

  • Указанная строка будет разбита на отдельные строки по разделителю ('\r\n'), после чего эти строки будут вставлены на место указанной строки (т.е. они будут располагаться перед ней).
  • Для вставки пустой строки добавить _T("\\r\\n").
Аргументы
[in]_SourceДобавляемая строка.
[in]_IndexИндекс строки, на место которой будут добавлены строки.
Исключения
std::exception
  • Индекс превышает количество строк в тексте.
