Набор вспомогательных библиотек

Проект утилиты обработки результата вычисления метрик кода в решении и генерации файлов отчетов для включения в документацию. Используется совместно с утилитой SourceMonitor в скрипте, запускаемом после сборки проекта на странице свойств .../Externals/Metrics.props.

Генерирует (в той же папке, что и исходные файлы) два файла:

  • Файл Metrics.result.html, содержащий результаты вычисления метрик для кода решения в целом (история 10 последних измерений).
  • Файл Metrics.result.warning.html, содержащий список самых проблемных мест в коде решения при последнем измерении для каждой метрики в отдельности.

Параметры запуска программы:

  • path (–path="path/to/...") - путь к папке, в которой располагаются файлы результата вычисления метрик кода в решении (Metrics.common.xml и Metrics.details.xml).
  • count (–count=10) - количество проблемных мест в коде, которое будет выводиться для каждой метрики (по умолчанию - 10).
Заметки
Для того, чтобы исключить из файла отчета определенные файлы, можно добавить в папку с исходными файлами файл с именем Metrics.exclude.txt, содержащий регулярные выражения исключаемых файлов (по одному в каждой строке).
