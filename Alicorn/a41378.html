Подробное описание

Класс входит в проект ToolsProcessingMetricsAppPage
Класс обработки детального результата вычисления метрик кода в решении.

  • Извлекает данные из файла Metrics.details.xml в указанной в конструкторе папке и формирует отчет (с перезаписью существующего) для включения в документацию решения.
  • Файл отчета (с именем Metrics.result.warning.html) создается в той же папке, что и исходный файл.
  • Файл отчета содержит списки самых проблемных мест в коде для каждой метрики в отдельности (количество записей - параметр конструктора).
  • В исходную папку можно добавить файл Metrics.exclude.txt с регулярными выражениями для файлов, которые должны игнорироваться.

Функция формирует сортированный список файлов для указанной метрики.

  • Функция предназначена для извлечения из общего набора вычисленных метрик файлов для указанной метрики.
Аргументы
[in]_MetricsИсходный набор метрик, прочитанный из файла отчета.
[in]_MetricIdИдентификатор метрики, для которой должен быть постоен список файлов.
[in]_TypicalValueТипичное значение метрики (сортировка будет производиться таким образом, что в начале списка будут идти файлы, у которых значение метрики максимально отличается (в абсолютном выражении) от указанного значения; если указано значение не равное нулю, файлы с нулевыми значениями метрик будут проигнорированы).
[in]_ExcludeFilesСписок регулярных выражений для файлов, которые должны игнорироваться.
Исключения
std::exception
