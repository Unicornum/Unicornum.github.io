Подробное описание

Класс входит в проект Document
Класс для чтения данных из файла, сохраненного в формате utf8 с сигнатурой.

  • Проверяет наличие сигнатуры.
  • Возвращает данные файла без сигнатуры.

◆ Utf8()

Utf8::Utf8 ( const Path_t &  _Path)
explicit

Конструктор чтения данных из файла.

  • Проверяет наличие сигнатуры utf8 в начале файла (0xEF, 0xBB, 0xBF).
  • Если сигнатура utf8 присутствует, читает данные с 3-й позиции.
Аргументы
[in]_PathПуть к исходному файлу.
Исключения
std::exception
  • Указанный файл не содержит сигнатуры utf8.
