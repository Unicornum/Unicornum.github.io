Класс входит в проект Image
Класс формирования данных изображения с указанными модификаторами.

  • Класс предназначен для формирования данных изображения в заданном формате.
  • 'Нормальным' форматом расположения пикселей в буфере (т.е. результатом, который получается без использования модификаторов) считается расположение байт, при котором отсчет начинается от верхнего левого угла изображения, а байты одного пикселя идут в порядке BGRA (0...3).
  • Без дополнительных преобразований (максимально быстро) формируется буфер, формат которого пригоден для непосредственного формирования текстуры OpenGL.
  • Поддерживаются следующие модификации изображения:
    • Размер пикселя (классы Pixel24, Pixel32).
    • Порядок следования компонентов цвета в пикселе (класс InvertPixel).
    • Порядок следования строк изображения (класс InvertLine).
    • Выравнивание строк изображения (класс Stride).
Параметры шаблона
TPixelФормат пикселя, в которых будет сформировано изображение (Pixel24, Pixel32).
