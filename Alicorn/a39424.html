Подробное описание

Раздел вспомогательных классов для использования в программах под Windows.

Запуск программы

Запуск программы без параметров и без ожидания ее завершения

::alicorn::system::platform::Process{ "calc.exe" }.Run();

Запуск программы с параметрами и ожиданием ее завершения

::alicorn::system::platform::Process{ "calc.exe" }.Wait(1000).Run({ uT("--help") });

Запуск программы с получением ее консольного вывода

::std::string ConsoleOutput;
auto Receiver = [&](const ::std::string & _Line)
{
ConsoleOutput += _Line + "\r\n";
};
::alicorn::system::platform::Process{ "calc.exe" }.StdOut(Receiver).Run();

◆ WINAPI_CHECK

#define WINAPI_CHECK   CHECK_PROXY(::alicorn::system::platform::WinapiCheckProxy)

Класс входит в Platform
Макрос предназначен для работы с функциями WinApi.

  • Подставленный перед вызовом функции, он произведет проверку возвращаемого значения и бросит исключение, если функция вернет FALSE.
  • Исключение будет в описании содержать место вызова функции (имя файла + номер строки), код ошибки и текстовое описание этой ошибки.

◆ ProcessOutput()

bool ProcessOutput ( const Poco::Pipe &  _Pipe,
Poco::ProcessHandle &  _hProcess,
const size_t  _PipeReadBufferSize,
process::ICallBack &  _CallBack 
)

Функция обработки консольного вывода программы.

  • Функция разбивает консольный вывод на строки и вызывает для каждой строки функцию Received() объекта обратного вызова.
Аргументы
[in]_PipeОбъект, из которого будем читать консольный вывод программы.
[in]_hProcessХэндл процесса, консольный вывод которого будет прочитан.
[in]_PipeReadBufferSizeРазмер буфера чтения информации из пайпа (за раз) в байтах (подробности см. у конструктора класса Process).
[in]_CallBackОбъект обратного вызова.
Возвращает
true Программа завершилась корректно.
false Программа была прервана принудительно, т.к. IsContinue() вернула false.
