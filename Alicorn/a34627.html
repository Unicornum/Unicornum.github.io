Подробное описание

Для предварительного объявления класса строки использовать:

#include <alicorn\std\string.forward.hpp>

Если объект строки должен содержаться в классе или требуется указать значение строки по умолчанию, для ускорения компиляции в заголовочный файл можно добавить только объявление класса:

#include <alicorn\std\string\string.hpp>

а уже в файл реализации - полное определение класса:

#include <alicorn\std\string.hpp>
#include <alicorn\std\string_cast.hpp>

Форматирование std::chrono::time_point

Форматирование даты/времени производится передачей вторым параметром строки форматирования.

Системное время

using namespace ::alicorn::extension::std;
const auto Now = ::std::chrono::system_clock::now();
const auto DateTimeText = string_cast<String>(Now, u8"%Y-%m-%d %H:%M:%S.%s %Z");
// В результате DateTimeText будет содержать строку вида
// "2017-05-12 11:02:03.123456 UTC+00.00", причем в том же формате, в котором
// была задана строка форматирования.

Локальное время

using namespace ::alicorn::extension::std;
const auto Now = ::std::chrono::system_clock::now();
const auto LocalTime = ::alicorn::extension::std::chrono::make_local_time(Now);
const auto DateTimeText = string_cast<String>(LocalTime, u8"%Y-%m-%d %H:%M:%S.%s %Z");
// В результате DateTimeText будет содержать строку вида
// "2017-05-12 14:02:03.123456 UTC+03.00", причем в том же формате, в котором
// была задана строка форматирования.

Функция входит в Std
Функция преобразования строк из одной кодировки в другую.

  • Функция представляет собой шаблон, предоставляющий интерфейс для преобразования строк класса String в/из объектов класса std::string в указанной кодировке, а также для изменения внутреннего формата хранения в объекте класса String.
  • Функция предназначена для поддержки работы класса alicorn::extension::std::String и позволяет производить следующие виды преобразований (для несуществующих будет брошено исключение):
  • String <-> std::string(1251).
  • String <-> std::string(866).
  • String <-> std::string(utf8).
  • String(utf8, ucs2, utf16, utf32) <-> String(utf8, ucs2, utf16, utf32).
  • String(utf8, ucs2, utf16, utf32) <- int.

Кодировка задается при помощи дочерних классов класса Locale.
