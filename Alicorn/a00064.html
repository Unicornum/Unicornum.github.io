Класс входит в Проект Converter
Класс, определяющий общий интерфейс для сжатия/распаковки данных.

Класс не предназначен для непосредсвенного использования, используются конкретные специализации шаблона класса:

  • Compress<Snappy> - сжатие/распаковка при помощи библиотеки Snappy.
Параметры шаблона
TImplementationОдин из следующих классов для специализации шаблона:
  • Snappy.
