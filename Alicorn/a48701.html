◆ GetNow()

DateTime::String_t DateTime::GetNow ( void )
static

Функция получения строки текущего времени.

  • Функция возвращает текущую дату/время в формате '[YYYY-MM-DD HH.MM.SS]'.
  • Возвращаемое этой функцией значение используется для формирования имени файла лога, поэтому оно должно содержать только допустимые для имен файлов знаки.
  • Возвращаемое этой функцией значение используется для формирования имени файла лога, от которого зависит механизм работы роллинга файлов лога, поэтому выбран именно такой порядок следования значений даты/времени.

Функция получения строки текущего времени, включая микросекунды и смещение UTC.

  • Функция возвращает текущую дату/время в формате '[YYYY-MMM-DD HH:MM:SS.MKS UTC+03.00]'.
