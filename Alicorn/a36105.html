Класс входит в проект Crypto
Класс для безопасного хранения пароля.

  • Хранит пароль в зашифрованном виде, предоставляя пароль как временный объект, который гарантированно затирает свое содержимое в памяти после разрушения.
  • Класс-наследник должен определить функцию DoDecrypt(), которая будет вызвана для каждого символа зашифрованного пароля.
Параметры шаблона
TCharТип элементов, в которых будет возвращен дешифрованный пароль.
TEncryptCharТип элементов, в которых храниться зашифрованный пароль.
