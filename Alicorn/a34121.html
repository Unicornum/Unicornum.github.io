Core::Data_t Core::GetData ( const Path_t &  _PathToFile) const

Функция получения данных указанного файла.

  • Поиск данных указанного файла будет производиться в том порядке, в котором реализации были добавлены в конструкторе.
Аргументы
[in]_PathToFileПуть к искомому файлу (интерпретация этого параметра у различных реализаций может быть различной, см. подробнее у конкретного класса реализации).
Возвращает
Data Бинарные данные указанного файла (один в один, т.е. размер буфера будет равен размеру файла).
Исключения
std::exception
  • Указанный файл не был найден ни одной из заданных в конструкторе реализаций.
