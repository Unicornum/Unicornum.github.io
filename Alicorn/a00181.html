Функция построения поддерева набора результатов прогона тестов.

  • Единственный объект информации о прогоне тестов добавляется в указанный узел без создания промежуточного узла.
Аргументы
[in]_TreeItemIdУзел дерева, в котором будет строится поддерево.
[in]_InfoОбъект с результатами прогона тестов.
[in]_IsFailedOnlyПризнак того, что в дерево должны войти только проваленные тесты.
[in]_IsSingleПризнак того, что объект _Info единственный в списке обектов.

Функция получения результата прогона теста для указанного узла дерева тестов.

  • Функция предназначена для использования в обработчике выбора мышью узла дерева для получения подробного описания результата прогона тестов.
Аргументы
[in]_TreeItemIdИдентификатор узла дерева, для которого требуется получить информацию о тесте.
Исключения
std::exception
  • Для указанного узла отсутствует связанная с ним информация.
