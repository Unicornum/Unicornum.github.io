Подробное описание

Класс входит в Settings
Класс настройкам раздела.

  • Предназначен для удобного доступа к настройкам раздела и дочерним разделам.
  • Объект раздела хранит объект корневого раздела, т.к. для работы класса реализации необходимо существование объекта корневого раздела все время, пока существует хотя бы один объект конкретного раздела настроек.

Функция для добавления особого параметра.

  • Назначение особых параметров - использование в качестве тегов для замены у других параметров.
  • Значения особых параметров можно установить только один раз и невозможно изменить (их можно только прочитать).
  • Особые параметры не будут сохранены в файле настроек.
Аргументы
[in]_NameИмя параметра.
[in]_ValueЗначение параметра.
Исключения
std::exception
  • Попытка добавить параметр, имя которого совпадает с именем уже существующего параметра.
