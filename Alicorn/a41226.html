◆ Run()

int Process::Run ( const Args_t & _Args = Args_t{})

Функция запуска программы.

  • Если не было задано время ожидания завершения программы и функция обратного вызова, то:
    • Функция запустит программу и сразу же вернет управление.
    • В качестве программы может быть задан путь к файлу (например, .txt), ярлыку или url, файл будет открыт в назначенной ему программе, ярлык или url будут открыты в браузере (аргументы командной строки в этом случае игнорируются).
    • В этом случае код возврата не имеет смысла).
  • Если было задано время ожидания завершения программы, то
    • Функция запустит программу и будет ожидать ее завершения указанное время.
    • Если программа не завершит работу за указанное время, она будет завершена принудительно.
    • Функция вернет управление, когда работа программы будет завершена.
  • Если была задана функция обратного вызова, то:
    • Функция запустит программу и будет читать ее консольный вывод (stdout и stderr).
    • После завершения работы программы (штатного или принудительного) консольный вывод будет передан по одной строке через функцию обратного вызова.
    • Функция вернет управление, когда работа программы будет завершена.
  • Программа с окном будет отображаться на экране, консольная программа будет запущена в скрытом режиме.
Аргументы
[in]_ArgsАргументы командной строки. См. также функцию ToParams().
Возвращает
Код, который вернула программа при завершении работы.
Исключения
std::exception
  • Указанной в конструкторе программы не существует.
  • Указанный в конструкторе файл существует, но не является запускаемой программой.
  • Было задано время ожидания завершения работы программы, но программа не завершила работу за указанное время.
Необходимо сделать

Запуск программы с ожиданием ее завершения и получением ее консольного вывода.

  • Функция запустит программу и будет читать ее консольный вывод (stdout и stderr).
  • Когда работа программы будет завершена, функция вернет управление.
  • После завершения работы программы (штатного или принудительного) консольный вывод будет передан по одной строке через функцию Received() объекта обратного вызова.
  • Для принудительного завершения работы программы функция IsContinue() объекта обратного вызова должна вернуть false.
  • Программа с окном будет отображаться на экране, консольная программа будет запущена в скрытом режиме.
Аргументы
[in]_ArgsАргументы командной строки (могут быть переданы как набор строк, где каждая строка - отдельный параметр, так и одной строкой, в которой параметры разделены пробелами).
Заметки
Использовать строки в формате UTF8.
Аргументы
[in]_CallBackОбъект обратного вызова, который будет получать консольный вывод запущенной программы.
[in]_IsReadFromAnotherThreadЗапуск чтения консольного вывода в отдельном потоке.
Возвращает
Код, который вернула программа при штатном завершении работы.
Исключения
std::exception
  • Указанной в конструкторе программы не существует.
  • Указанный в конструкторе файл существует, но не является запускаемой программой.
Process::KilledException
  • Программа была принудительно завершена возвратом false функцией объекта обратного вызова.

std::string Process::Tokenize ( const ::std::string & _Source,
process::ICallBack & _CallBack )
static

Функция разбиения получяемой строки на отдельные строки.

  • Если исходная строка содержит строки, каждая из которых завершается символами переноса, то все эти строки будут переданы объекту обратного вызова и функция вернет пустую строку.
  • Если исходная строка завержается не символами переноса, то объекту обратного вызова будут переданы все строки, кроме завершающей и функция вернет остаток.

◆ ToParams()

template<class TData >
Process::String_t alicorn::system::platform::Process::ToParams ( const TData & _Data)

Функция преобразования объекта класса в строку для передачи его вызываемой программе в качестве параметра.

  • Для того, чтобы объект класса можно было преобразовать в строку параметра, он должен поддерживать сериализацию при помощи Boost.Serialization, т.е. содержать функцию
    template<class TArchive>
    void serialize(TArchive & _Archive, const unsigned int)
    {
    _Archive & BoolValue; // Для каждой переменной класса
    }
  • В сериализованном объекте все кавычки будут заменены на специальную группу символов для того, чтобы можно было передавать строки, содержащие кавычки без разрыва параметра командной строки на отдельные строки.
  • Сериализованный объект будет обрамлен кавычками для того, чтобы при его передаче вызываемая программа получила его как единый параметр (вызываемая программа получит его без обрамляющих кавычек).
  • Для обратного преобразования строки в объект класса использовать функцию FromParams().
Аргументы
[in]_DataИсходный объект.
Возвращает
Строка-параметр, пригодная для передачи вызываемой программе.
Исключения
std::exception
  • Действие невозможно (подробнее см. описание исключения).
