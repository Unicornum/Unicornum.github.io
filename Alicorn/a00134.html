Подробное описание

Класс входит в Проект Std
Класс-обертка для строк из стандартной библиотеки.

Обеспечивает:

  • Совместимость с 'сырыми' строками, объявленными через макрос _T(), который переопределяется таким образом, чтобы соответствовать внутреннему содержимому класса (когда в Visual Studio станет доступен префикс U, следует переопределить _T(), а UsingString_t определить через std::u32string).
  • Удобный интерфейс.
  • Допускает предварительное объявление (использовать файл string.forward.hpp).

При изменении класса строки проверить работу функций/классов:

  • alicorn::extension::std::string_cast().

Примеры использования:

Исходные данные:

using String_t = ::alicorn::extension::std::String;
using StdString_t = String_t::String_t;
using Char_t = String_t::Char_t;
const Char_t * m_RawWString = nullptr;
StdString_t m_WString;
m_RawWString = _T("Текст Text");
m_WString = m_RawWString;

Конструирование объектов:

String_t Example1 = m_RawWString;
String_t Example2(m_RawWString);
String_t Example1 = m_WString;
String_t Example2(m_WString);
String_t Example = m_RawWString;
String_t Copy1 = Example;
String_t Copy2(Example);

Копирование объектов:

const String_t Source = m_RawWString;
String_t Example;
Example = Source;

Количество символов в строке:

String_t Example1 = m_RawWString;
const String_t::Size_t Lenght = Example1.GetLenght();

Сравнение строк:

const bool IsEmpty = (Example1 == String_t::Empty()); // false

String_t Example2 = _T("абв");

EXPECT_FALSE(Example2 == Example1);
EXPECT_FALSE(Example1 == Example2);
EXPECT_TRUE(Example2 == Example2);

EXPECT_TRUE(Example3 == Example2);
EXPECT_TRUE(Example1 == Example5);
EXPECT_FALSE(Example2 == Example5);

String_t Example2 = _T("абв");

Удаление пробельных символов:

String_t TrimmedSource = Source.Trim(); // m_RawWString

Выделение подстроки:

String_t Result2 = Source.SubString(3); // "Текст Text"

Сложение строк:

Поиск подстроки:

String_t Source = _T("123") + String_t(m_RawWString);
String_t Text = _T("Текст");
const String_t::Size_t Find1 = Source.Find(_T("Текст")); // 3
const String_t::Size_t Find2 = Source.Find(Text); // 3
const String_t::Size_t Find3 = Source.Find(_T("Текст"), 5); // String_t::Npos
const String_t::Size_t Find4 = Source.Find(_T("Unknown")); // String_t::Npos

Замена подстроки:

String_t Source = _T("АБВ ABC АБВ ABC АБВ");
auto Expected = _T("А** ABC А** ABC А**");
String_t Result1 = Source.Replace(_T("БВ"), _T("**")); // Expected
String_t Pattern = _T("БВ");
String_t Result2 = Source.Replace(Pattern, _T("**")); // Expected
String_t Change = _T("**");
String_t Result3 = Source.Replace(_T("БВ"), Change); // Expected
String_t Result4 = Source.Replace(Pattern, Change); // Expected

Приведение к нижнему регистру:

String_t Source = _T("АБВ ABC");
auto Expected = _T("абв abc");
String_t Result = Source.ToLower(); // Expected

Разбиение на подстроки:

String_t Source = _T("АБВ ABC АБВ ABC АБВ ");
auto Separator = _T(" ");
const ::std::vector<String_t> Result = Source.Tokenize(Separator);
// Result:
// "АБВ"
// "ABC"
// "АБВ"
// "ABC"
// "АБВ"

Методы

std::vector< String > String::Tokenize ( const String &  _Separator) const

Функция для разбиения строки.

  • Разбиение производится по набору символов, точно совпадающих с указанным параметром (сама строка-разделитель в результат не попадет).
Аргументы
[in]_SeparatorСтрока-разделитель.
Возвращает
Набор строк, на которые была разбита исходная строка.
