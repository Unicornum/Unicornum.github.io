Набор вспомогательных библиотек

Описание решения

Решение представляет собой набор проектов универсальных классов, которые могут быть использованы в различных проектах.

Содержит классы

  • Расширяющие возможности стандартной и внешних библиотек.
  • Взаимодействия программы с операционной системой.
  • Доступа к данным, которые будет использовать программа.
  • Полнофункциональных модулей, созданных для выполнения конкретной задачи.
  • Вспомогательных утилит.

Платформы

Windows

  • Собирается при помощи Visual Studio 2015 (x86, x64).
  • Тестируется в Windows 7 x64.

Android

  • Собирается при помощи Visual Studio 2015 (ARM).
  • Тестируется на устройстве с API15 ARM.

Версии используемых утилит

Android compiler: arm-linux-androideabi-gcc.exe (GCC) 4.9
Android stl library: gnustl_static
Android API Level: android-19
Android SDK build tools: 27.0.3
Android SDK platform: android-27
Android NDK: crystax-ndk-10.3.2
JDK: 1.8.0_144
Apache Ant: 1.10.2
