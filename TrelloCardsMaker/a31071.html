Требования первого этапа (версия 0.1.0)

Программа должна:

  • Корректно работать (масштабироваться) при любых ориентации/разрешениях экрана используемого устройства.
  • Позволять вводить текст создаваемой карточки при отсутствии доступа в Интернет с автоматической отправкой карточек при появлении такого доступа.
  • Хранить неотправленные карточки в месте, доступном в 'ручном' режиме.
  • Хранить неотправленные карточки в формате, пригодном для вставки в Trello вручную (в случае необходимости).
  • Изменения данных программы должны сохраняться таким образом, чтобы при нажатии кнопки 'Home' в любой момент времени не происходила потеря изменений.

Окно создания карточки

Содержит:

  • Многострочное поле ввода текста создаваемой карточки.
  • Кнопку 'Отправить', при нажатии которой карточка с указанным текстом отправляется, а поле ввода текста очищается (чтобы можно было сразу вводить текст следующей карточки).
  • При нажатии стандартной кнопки 'Назад' программа завершает работу.

Далее...

Программа должна:

  • Предоставлять выбор доски Trello, для которой создается карточка.
  • Позволять добавлять/настраивать/удалять доски Trello для отправки карточек.
  • При нажатии стандартной кнопки 'Назад' в окне создания карточки происходит переход к окну выбора доски Trello.

Окно выбора доски Trello

  • Начальное окно при запуске/активации программы.
  • При нажатии стандартной кнопки 'Назад' программа должна возвращать управление Android'y, оставляя рабочий поток, пытающийся отправлять неотправленные карточки в случае их наличия.

Содержит:

  • Кнопку создания новой доски Trello (при нажатии которой программа переключается на окно создания доски Trello).
  • Кнопку 'Архив досок' при нажатии которой программа переключается на окно списка архивированных досок.
  • Доступные доски Trello в виде списка в визуально отделенном от кнопок создания доски/архива разделе.

Строка списка выбора доски Trello

  • Кнопка с именем доски Trello (при нажатии кнопки программа переключается на окно создания карточки).
  • Кнопка редактирования доски Trello (при нажатии кнопки программа переключается на окно создания доски Trello).
  • Кнопка удаления доски Trello (при нажатии кнопки доска переноситься в архивный список).

Окно создания доски Trello

Содержит:

  • Поле ввода имени доски.
  • Поле ввода почты, на которую будут отправляться карточки.
  • Кнопку принятия изменений, при нажатии которой программа должна переключиться на окно выбора доски Trello с внесением изменений.
  • При нажатии стандартной кнопки 'Назад' программа должна переключиться на окно выбора доски Trello без внесения изменений.
  • Если кнопка 'Назад' была нажата после создания новой доски, созданная доска 'по умолчанию' должна быть удалена.

Окно списка архивных досок Trello

  • Содержит архивные доски в виде списка конопок, при нажатии которых выбранная доска переносится в список досок, доступных для выбора в окне выбора доски Trello.
  • При нажатии стандартной кнопки 'Назад' программа переключается на окно выбора доски Trello.
