Подробное описание

Класс входит в проект Model
Реализация класса хранения данных модели в файле.

  • Класс не хранит никаких реальных данных (все функции используют запись/чтение файла) и не может иметь никакого состояния.
  • Все функции класса не бросают исключений, описания возникающих ошибок записываются в список, содержимое которого может быть получено при помощи функции GetErrors().

Функция сохранения списка карточек в файл.

  • Сохранение производится через создание промежуточного файла, чтобы исключить повреждение файла со списком карточек в случае, когда новая версия программы устанавливается в тот момент, когда сохраняется изменившийся список.
