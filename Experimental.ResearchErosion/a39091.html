Подробное описание

Used for simulating a rainfall event on a terrain. Stores information on the terrain that is required to simulate the effect of hydraulic erosion.

Заметки
The class should only be used once per rainfall event.

Called at the beginning of each iteration. This function will compute the flow rate of each cell, based on the level of water and the elevation at each point in the height map.

Аргументы
heightThe function taking an x and y coordinate and returning the height value at each point in the map.
waterThe function taking an x and y coordinate and returning the water level at each point in the map. The water levels can be initialized to simulate rainfall or river streams.

Erodes and deposites sediment, and then moves remaining sediment based on the velocity of the water at each cell.

Аргументы
kCA function taking an x and y coordinate and returning the carry capacity constant at that particular location.
kDA function taking an x and y coordinate and returning the deposition constant at that particular location.
kEA function taking an x and y coordinate and returning the erosion constant at that particular location.
heightAdderA function taking an x and y coordinate, as well as a height delta, and adding the value to the height model.
Заметки
For simple models, Kc Ke and Kd can both be single, uniform values.
