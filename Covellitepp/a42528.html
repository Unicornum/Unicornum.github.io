auto Configuration::GetSystemLanguage ( void  ) const

Функция получения текущего системного языка.

  • Назначение функции - получение строки, которую можно непосредственно использовать для построения имени файла, содержащего локализованные строки, используемые программой.
Возвращает
"English" Для английского языка.
"Russian" Для русского языка.
"Default" Если определить язык системы не удалось.
