Подробное описание

Класс входит в проект Covellite.Rocket
Класс окна, отрисовыващия GUI при помощи libRocket.

  • Является оберткой для Rocket::Core::Context и инкапсулирет в себе взаимодействие libRocket и событий Android (касание экрана, ввод текста, отрисовка элементов управления, полусение событий от libRocket).

void Window::LoadFontFace ( const Path_t &  _PathToFontFile)
staticprotected

Функция загрузки используемых шрифтов.

  • Класс-наследник должен перед созданием слоев вызвать эту функцию для используемых в rml-файлах шрифтов.
  • Функция может быть вызвана несколько раз для загрузки несколких шрифтов.
  • Ошибка (если таковая возникнет) будет записана в лог через интерфейс System.
Аргументы
[in]_PathToFontFileПуть к файлу шрифта относительно папки assets.
