◆ Input

enum covellite::Event::Input

Группа сообщений ввода пользователя.

Элементы перечислений
Motion 

Сигнал вызывается при движении пальца по экрану (параметр - params::Motion).

KeyPressed 

Сигнал вызывается при нажатии кнопки на виртуальной клавиатуре (параметр - params::KeyPressed). 32 - это код, который генерирует java код при нажатии кнопки на клавиатуре (понадобилось из-за того, что в нативном коде невозможно получить код нажатой клавиши за пределами ASCII кодировки).

KeyUp 

Сигнал вызывается при отпускании кнопки на виртуальной клавиатуре (параметр - params::KeyUp).

◆ Window

enum covellite::Event::Window

Группа оконных сообщений.

Элементы перечислений
Redraw 

Требуется перерисовка окна приложения, например, при повороте устройства, когда меняются размеры главного окна программы (без параметров).

Draw 

Сигнал вызывается при необходимости перерисовки окна. В зависимости от способа запуска программы (см. Application::Run()), будет вызываться непрерывно, либо только после получения программой сообщения (без параметров).
