  • Во время парсинга .rml файла все строки (такие как идентификаторы, атрибуты и значения xml узлов) будут передаваться функции Translate().

virtual Utf8String_t covellite::rocket::IStringTranslator::Translate ( const Utf8String_t &  )
pure virtual

Функция преобразования 'сырой' строки (прочитанной из .rml файла) в строку, которая будет отображаться на экране (преобразовние предназначено, в первую очередь, для локализации приложений).

Замещается в covellite::rocket::Window и ExampleWindow.
