LostFocus 

Приложение потеряло фокус ввода (без параметров).

GainedFocus 

Приложение получило фокус ввода (без параметров).

Resize 

Изменение размеров окна (без параметров).

Изменение размеров окна программы, например, при повороте Android устройства (без параметров).

Drawing 

Сигнал вызывается при необходимости перерисовки окна. В зависимости от способа запуска программы (см. Application::Run()), будет вызываться непрерывно, либо только после получения программой сообщения (без параметров).

StartDrawing 

Сигнал вызывается непосредственно перед событием Drawing (без параметров).

FinishDrawing 

Сигнал вызывается сразу после события Drawing (без параметров).

Close 

Закрытие окна приложения (без параметров).

Resize 

Изменение размеров окна (без параметров).
