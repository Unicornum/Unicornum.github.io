◆ Is()

bool KeyUp::Is ( ControlKey  _ControlKey) const
inline

Функция проверки соответствия 'сырого' кода нажатой кнопки коду управляющей кнопки.

  • Используется для распознавания нажатия управляющих кнопок Android устройства.
Аргументы
[in]_ControlKeyОдно из значений перечисления ControlKey.
Возвращает
true AsciiKeyCode соответствует коду указанной кнопки.
false AsciiKeyCode не соответствует коду указанной кнопки.
Необходимо сделать:
"Back: Alt+Left" ;
