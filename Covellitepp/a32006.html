template<class TEventType >
bool covellite::core::Events::DoCommand ( TEventType  _Event,
const Params &  _Params 
)
inline

Функция активации события.

  • Если во время активации указанного события происходит исключение, оно будет перехвачено и будет активирован сигнал Event::Error, в качестве парамера которому будет передано описание возникшего исключения.
Аргументы
[in]_EventИдентификатор активируемого события.
[in]_ParamsПараметры, передаваемые активируемому событию.
Возвращает
true
  • Событие было активировано удачно.
false
  • Во время обработки события произошло исключение.
